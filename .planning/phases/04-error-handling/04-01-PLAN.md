구현 사항:
1. LogLevel enum: "debug" | "info" | "warn" | "error"
2. logger 객체 export with methods: debug, info, warn, error
3. 각 메서드는 (message: string, context?: Record<string, unknown>) 시그니처
4. 출력 포맷: JSON.stringify({ level, message, timestamp: new Date().toISOString(), ...context })
5. NODE_ENV === "development"일 때는 console 메서드 사용 (가독성), production에서는 JSON 포맷
6. error 레벨에서는 stack trace도 포함 (context.error가 Error 인스턴스일 때)

예시 사용:
```typescript
logger.info("Search query", { query: "강남", results: 5 });
logger.error("DB query failed", { error: e, route: "/api/search" });
```
