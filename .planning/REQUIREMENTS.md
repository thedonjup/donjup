# Requirements: v1.1 데이터 분석 고도화

**Defined:** 2026-03-28
**Core Value:** 사용자가 아파트 시장 흐름을 정확히 읽고, 신뢰할 수 있는 데이터 기반 의사결정을 할 수 있게 하는 것.

## 가격 정규화 (NORM)

- [x] **NORM-01**: 차트에서 "전체 면적" 합산 그래프를 제거하고, 면적별로만 가격 추이를 표시한다
- [ ] **NORM-02**: 저층(3층 이하 또는 전체층수 하위 20%) 거래를 고층 기준가로 환산하여 차트에 반영한다 (층별 보정계수: 1층 -13%, 2층 -10%, 3~4층 -4%)
- [x] **NORM-03**: 차트의 추이선을 단순 실거래가가 아닌 3개월 이동중위가 기반으로 표시한다
- [x] **NORM-04**: 이상거래(직거래 + 시세 대비 30% 이상 저가)를 차트에서 자동 필터링한다
- [ ] **NORM-05**: 사용자가 "저층 포함/제외" 토글로 저층 거래 표시 여부를 선택할 수 있다

## 차트 개선 (CHART)

- [ ] **CHART-01**: 기간 선택 탭(1개월/3개월/6개월/1년/전체)을 차트에 추가한다
- [ ] **CHART-02**: 면적별 차트에서 매매가 + 전세가를 동시에 표시한다 (듀얼 라인)
- [ ] **CHART-03**: 전세가율(매매가 대비 전세 비율) 오버레이 라인을 차트에 추가한다

## 랭킹 정교화 (RANK)

- [ ] **RANK-01**: 폭락/신고가 변동률을 층별 보정 후 가격 기준으로 산출한다 (저층 노이즈 제거)
- [ ] **RANK-02**: 이상거래(직거래 저가, 친족 거래 추정)를 랭킹 산정에서 자동 제외한다
- [ ] **RANK-03**: 랭킹 리스트에서 저층 거래는 "저층" 라벨을 표시하되, 변동률은 고층 환산 기준으로 산출한다

## 지역 지수 (INDEX)

- [ ] **INDEX-01**: 지역 군집(강남3구, 마용성, 노도강, 수도권 주요 등)을 정의하고 군집별 중위가 지수를 산출한다
- [ ] **INDEX-02**: 군집별 지수의 시계열 차트를 제공한다 (기준시점 = 데이터 최초일 100)
- [ ] **INDEX-03**: 시도/시군구 단위 평균 매매가·중위가를 지역별 시세 페이지에 표시한다
- [ ] **INDEX-04**: 군집 지수 페이지를 신규 생성하여 S&P500 스타일 대시보드를 제공한다

## 금리 표현 (RATE)

- [ ] **RATE-01**: 금리 초기 화면에 시중금리 평균값 1개만 대표로 표시한다 (은행별 주담대 평균)
- [ ] **RATE-02**: 기준금리·COFIX 등 세부 지표는 펼침(accordion) 또는 상세 페이지로 이동한다
- [ ] **RATE-03**: 은행별 상세 금리는 터치/클릭 시 확장되는 형태로 변경한다

## 전세가율/갭 분석 (GAP)

- [ ] **GAP-01**: 아파트 상세 페이지에 면적별 전세가율(매매가 대비 전세 비율)을 표시한다
- [ ] **GAP-02**: 아파트 상세 페이지에 갭 금액(매매가 - 전세가)을 표시한다
- [ ] **GAP-03**: 전세가율 추이 차트를 면적별로 제공한다
