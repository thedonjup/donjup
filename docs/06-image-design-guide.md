#### 1-1. 앱 아이콘 (App Icon)

| 항목 | 내용 |
|------|------|
| **이미지명** | `app-icon-512.png` |
| **용도** | PWA 설치 아이콘, 앱스토어 아이콘, manifest.json icons |
| **크기/규격** | 512x512px, PNG, 투명 배경 없음 (maskable 대응) |
| **설명** | 녹색(#059669) 라운드 사각형 배경 위에 흰색 ₩ 심볼. 안전 영역(safe zone) 고려하여 중앙 70% 영역에 심볼 배치. 앱 아이콘 그리드에 맞춘 비율. |

**AI 프롬프트:**

> **프롬프트 1 (플랫 디자인):**
> 모바일 앱 아이콘 디자인, 512x512 픽셀, 정사각형. 배경색은 에메랄드 그린(#059669) 단색으로 채운 라운드 사각형(모서리 반경 약 23%). 중앙에 흰색(#FFFFFF) 원화 기호(₩)를 굵은 산세리프체로 크게 배치. 플랫 디자인 스타일, 그림자 없음, 그라데이션 없음. 미니멀하고 깔끔한 핀테크 앱 느낌. 배경은 단색으로 가득 채우고 여백 없이.

> **프롬프트 2 (그라데이션 버전):**
> 모바일 앱 아이콘, 512x512px 정사각형. 라운드 사각형 프레임 안에 좌상단에서 우하단으로 향하는 그라데이션 배경(#10b981에서 #059669를 거쳐 #047857로). 중앙에 흰색 ₩ 기호를 두껍고 모던한 산세리프 서체로 배치. 약간의 입체감을 주는 미세한 내부 그림자. iOS/Android 앱 아이콘 가이드라인 준수. 금융 테크 앱 분위기.

> **프롬프트 3 (심볼+텍스트 결합):**
> 앱 아이콘 디자인, 512x512px. 짙은 에메랄드 그린(#059669) 라운드 사각형 배경. 상단 60%에 큰 흰색 ₩ 기호, 하단 25%에 "돈줍"이라는 한글 텍스트를 흰색 볼드체로 작게 배치. 전체적으로 미니멀한 핀테크 브랜드 아이콘 스타일. 텍스트와 심볼 사이 적절한 여백. maskable 아이콘 safe zone 내에 모든 요소 배치.

| 항목 | 내용 |
|------|------|
| **이미지명** | `favicon.svg` / `favicon-32.png` |

| **설명** | 앱 아이콘의 축소 버전. 32px에서도 ₩ 심볼이 명확히 보여야 함. 복잡한 디테일 제거. |

**AI 프롬프트:**

> **프롬프트 1:**
> 웹 파비콘 디자인, 32x32 픽셀, 초소형 아이콘. 에메랄드 그린(#059669) 정사각형 배경에 흰색 ₩ 기호. 극도로 단순화된 형태, 1~2px 수준의 디테일만 사용. 작은 크기에서 선명하게 보이도록 두꺼운 획. 픽셀 퍼펙트 디자인.

> **프롬프트 2:**
> 미니멀 파비콘, 32x32px. 둥근 정사각형 안에 ₩ 심볼만 있는 초간결 디자인. 배경 #059669, 심볼 #FFFFFF. 웹 브라우저 탭에서 다른 사이트와 구별되도록 선명한 대비. SVG 스타일의 깔끔한 벡터 느낌.

> **프롬프트 3:**
> 파비콘용 아이콘, 32x32px 및 16x16px 대응. 녹색 원형 또는 라운드 사각형 배경에 흰색 ₩. 배경색 #059669. 저해상도에서도 식별 가능한 굵은 획의 심볼. 심플한 2색 구성, 안티앨리어싱 최소화.

**AI 프롬프트:**

> **프롬프트 1 (도시 야경 배경):**
> 인스타그램 카드뉴스 표지 배경, 1080x1350px 세로형(4:5 비율). 서울 아파트 단지의 야경을 촬영한 듯한 이미지. 수많은 아파트 창문에서 불빛이 새어나오는 어두운 밤 풍경. 전체적으로 어두운 블루-네이비 톤(#0f172a 계열). 상단 30%와 하단 20%에 짙은 그라데이션 오버레이가 깔려 텍스트를 올릴 수 있는 구조. 분위기는 고요하면서도 긴장감 있는 도시 야경. 사실적인 사진 스타일.

> **프롬프트 2 (추상적 데이터 비주얼):**

#### OG/소셜 공유 이미지
```
1차: DALL-E 3로 기본 컨셉 이미지 생성
2차: Ideogram으로 한글 텍스트 포함 버전 제작
3차: Figma/Canva에서 최종 정리 및 텍스트 교정
```

### 3.3 비용 효율적 제작 우선순위

### 3.4 프롬프트 작성 팁

1. **색상 지정은 HEX 코드로**: "녹색"보다 "#059669 에메랄드 그린"이 정확
2. **크기를 명시**: "1080x1350px 세로형 4:5 비율"처럼 구체적으로
3. **텍스트 영역 확보 지시**: "중앙에 텍스트를 올릴 수 있도록 비워둘 것"
4. **네거티브 프롬프트 활용**: "사람 없음, 워터마크 없음, 텍스트 없음"
5. **스타일 참조**: "핀테크 앱 대시보드 스타일", "금융 데이터 시각화"
6. **한글 텍스트 포함 시**: Ideogram 사용, 텍스트 정확도 확인 필수
7. **일관성 유지**: 시리즈물은 첫 이미지의 시드/스타일을 저장하여 동일 톤 유지
