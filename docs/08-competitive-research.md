## 1. 호갱노노

> **"아파트 실거래가 1등 앱"** — 직방 자회사, 2015년 창업

### 1-1. 핵심 기능

| 기능 | 설명 |
|------|------|
| **실거래가 조회** | 국토교통부 데이터 기반, 지도 위에 단지별 실거래가를 직관적으로 표시 |
| **시세 변동 추적** | 1개월 이내 호가/시세 변동을 그래프로 시각화, 신고가·최저가 알림 |
| **학군 정보** | 초·중·고 학군 경계, 학업성취도, 학원가 정보 통합 제공 |
| **3D 일조량 분석** | 시간대·계절별 세대별 일조량을 3D로 시뮬레이션 |
| **입주물량** | 지역별 신규 입주 예정 물량 캘린더 |
| **AI 중개사** | 대화형으로 조건에 맞는 단지 추천 (직방 데이터 연동) |
| **3D 단지투어** | 단지 전경, 동별 조망, 미분양·신축 단지 가상 임장 |
| **중개사 채팅** | 전화 대신 앱 내 중개사 직접 채팅 |
| **안심 알리미** | 등기 변동 알림 (전세사기 방지) |
| **커뮤니티 리뷰** | 실거주민이 작성하는 단지별 리뷰 (플랫폼 내 가장 활성화된 기능) |

### 1-2. 정보 계층

```
[지도 뷰 (최상위)]
  └── 지역 선택 (시/도 → 구/군)
        └── 단지 리스트 (가격순, 면적순, 인기순 등)
              └── 단지 상세
                    ├── 시세/실거래가 그래프
                    ├── 평형별 매물
                    ├── 학군 정보
                    ├── 주변 인프라
                    ├── 일조량 3D
                    ├── 거주민 리뷰
                    └── 중개사 연결
```

### 1-3. 차별화 포인트

- **커뮤니티 기반 리뷰**: 실거주민의 솔직한 후기가 핵심 경쟁력. 다른 플랫폼 대비 압도적인 리뷰 양과 질
- **극도로 낮은 이탈률**: 21.8%로 업계 최저 (직방 38.3%, 다방 54.7%)
- **장기 체류**: 평균 사용기간 519일로 직방(379일)의 1.4배
- **다양한 지도 레이어**: 신고가, 인구변동, 외지인비율, 거주직장인 연봉, 경사도, 출근시간, 개발호재, 상권 등 20개 이상의 데이터 레이어를 지도 위에 오버레이

### 1-4. 사용자 경험

**장점:**
- 시각적으로 직관적인 지도 기반 UI
- 학군 + 생활인프라까지 원스톱 확인 가능
- 리뷰 기능으로 "살아본 사람"의 이야기를 들을 수 있음
- 일조량 3D 분석은 타 플랫폼에 없는 독보적 기능

**단점:**
- 아파트 중심이라 빌라, 단독주택 정보 부족
- 중개 거래 자체는 직방으로 연결됨 (독립적 거래 불가)
- 정보 레이어가 너무 많아 초보자에겐 복잡할 수 있음

### 1-5. 수익 모델

| 수익원 | 설명 |
|--------|------|
| **중개사 노출 광고** | 노출 비율 기반 과금 (타사 월정액과 차별화) |
| **대출 연계** | 수요자 대출상담 → 은행 직원 매칭 → 수수료 |
| **직방 통합 광고상품** | 직방/호갱노노 통합 광고 패키지 (분양 광고 등) |
| **프리미엄 데이터** | 아파트PRO, 아파트BASIC 등 중개사 전용 유료 서비스 |

### 1-6. 데이터 소스

- 국토교통부 실거래가 공개시스템
- 교육부 학군 데이터
- 행정안전부 인구이동 통계
- 자체 크롤링 (매물 호가, 개발호재 등)
- 사용자 생성 콘텐츠 (거주민 리뷰)
- 직방 매물 데이터 연동

### 1-7. 인기 기능 (사용자 이용 빈도순)

1. 실거래가 조회 + 시세 그래프
2. 거주민 리뷰 열람
3. 학군 정보 확인
4. 지도 기반 탐색 (각종 레이어)
5. 3D 일조량 분석

## 2. 직방

> **"No.1 부동산 앱"** — 원·투룸에서 아파트까지 확장, 호갱노노 모회사

### 2-1. 핵심 기능

| 기능 | 설명 |
|------|------|
| **매물 검색** | 아파트, 오피스텔, 원룸, 투룸, 빌라 등 전 주거 유형 커버 |
| **지도 기반 검색** | 위치/역 중심 매물 탐색, 필터링(가격, 면적, 층수 등) |
| **3D 단지투어** | VR 기반 아파트 단지 외부 + 내부 투어 |
| **VR 홈투어** | 가상현실로 실내 인테리어까지 확인 |
| **AI 중개사** | 24시간 대화형 단지 추천 서비스 |
| **분양 정보** | 신규 분양 단지 정보 및 청약 일정 |
| **스마트홈** | 직방 스마트홈 IoT 연동 (B2C + B2B) |
| **중개사 플랫폼** | 중개사 전용 매물관리 + 광고 시스템 |

### 2-2. 정보 계층

```
[메인 홈]
  ├── 원룸/투룸 (지도 기반 매물 리스트)
  ├── 아파트 (호갱노노 데이터 연동)
  │     ├── 단지 정보
  │     ├── 3D 단지투어 / VR 홈투어
  │     └── AI 중개사 추천
  ├── 오피스텔
  ├── 빌라
  └── 분양
```

### 2-3. 차별화 포인트

- **전 주거유형 통합**: 원룸 ~ 아파트까지 한 앱에서 해결
- **VR/3D 기술 선도**: 업계 최고 수준의 가상 임장 경험
- **호갱노노 시너지**: 아파트 데이터는 호갱노노의 축적된 DB 활용
- **스마트홈 사업 확장**: 부동산 플랫폼을 넘어 주거 IoT까지 영역 확대

### 2-4. 사용자 경험

**장점:**
- 원룸/오피스텔은 매물량이 압도적
- VR 투어가 실제 임장을 대체할 수 있는 수준
- 직관적인 UI/UX

**단점:**
- 아파트 정보는 호갱노노와 중복 (앱을 두 개 써야 하는 느낌)
- 허위 매물 이슈가 과거부터 꾸준히 제기됨
- 2024년까지 적자 지속 (2025년 턴어라운드 조짐)

### 2-5. 수익 모델

| 수익원 | 설명 |
|--------|------|

| **분양 정보 광고** | 건설사/시행사 대상 정보성 광고 상품 |
| **아파트PRO/BASIC** | 지역 중개사 전용 유료 서비스 |

### 2-6. 데이터 소스

## 3. 네이버 부동산

> **"네이버페이 부동산"** (2023년 6월 명칭 변경) — 국내 최대 트래픽 부동산 플랫폼

### 3-1. 핵심 기능

| 기능 | 설명 |
|------|------|
| **매물 통합 검색** | 아파트, 빌라, 오피스텔, 상가, 토지 등 모든 부동산 유형 |
| **지도 기반 검색** | 네이버 지도 연동, 주변 인프라(교통, 학교, 편의시설) 통합 표시 |
| **실거래가 그래프** | 공공데이터 기반 거래 가격 추이 그래프 |
| **분양/청약 정보** | 분양 일정, 청약 캘린더, 분양가 정보 |
| **시세 비교** | 주변 단지 대비 시세 비교 분석 |
| **대출 정보** | 대출 관련 정보 및 계산기 연동 |
| **매물 진위 확인** | 집주인 의뢰 여부를 전화/서류로 확인하는 시스템 |
| **개발 호재 오버레이** | 지하철 노선 예정지, 학군 경계선 등 지도 오버레이 |

### 3-2. 정보 계층

```
[네이버 검색 / 네이버 부동산 홈]
  ├── 지도 기반 매물 탐색
  │     ├── 매물 상세 (가격, 면적, 층, 향, 사진)
  │     ├── 실거래가 탭 (그래프)
  │     └── 주변 시세 비교
  ├── 분양/청약 정보
  ├── 뉴스/시황
  └── 커뮤니티 (부동산 카페 연동)
```

### 3-3. 차별화 포인트

- **압도적 트래픽**: 네이버 검색 유입으로 자연스러운 트래픽 확보 (부동산 검색의 시작점)
- **매물 신뢰도**: 집주인 의뢰 확인 시스템으로 허위매물 최소화
- **생태계 연동**: 네이버 카페(부동산 커뮤니티), 네이버 페이(결제), 네이버 지도 등과 자연스럽게 연동
- **아실 인수(2025)**: 네이버페이가 300억 원에 아실을 인수하여 실거래가 데이터 분석 역량 대폭 강화
- **정보회사 생태계**: 이실장, 매경부동산, 아실, 피터팬 등 다수 정보회사와 연계

### 3-4. 사용자 경험

**장점:**
- "부동산" 검색 시 네이버에서 바로 접근 가능 (별도 앱 불필요)
- 매물 종류가 가장 다양 (상가, 토지까지)
- 허위매물 필터링 시스템
- 네이버 카페 커뮤니티와 자연스러운 연결

**단점:**
- 2024년 8월 독립 앱 서비스 종료 (네이버 앱 내 통합)
- 아파트 전문 분석 깊이는 호갱노노/아실 대비 얕음
- 데이터 시각화 수준이 경쟁사 대비 단순

### 3-5. 수익 모델

| 수익원 | 설명 |
|--------|------|
| **중개사 매물 광고** | 정보회사를 통한 매물 등록 시 광고비 |
| **프리미엄 노출** | 상위 노출 광고 상품 |
| **분양 광고** | 건설사 대상 분양 홍보 광고 |
| **네이버 검색 광고 연동** | 부동산 키워드 검색 광고 |

### 3-6. 데이터 소스

- 국토교통부 실거래가
- 정보회사 연계 매물 데이터
- 서울시 등 지자체 데이터 공유 협약
- 네이버 지도 데이터
- 아실 빅데이터 (인수 후 통합)
- 사용자 생성 콘텐츠 (네이버 카페)

### 3-7. 인기 기능

1. 지도 기반 매물 검색
2. 실거래가 조회
3. 분양/청약 일정 확인
4. 주변 시세 비교

### 4-2. 정보 계층

```
[메인 홈 / 지도 뷰]
  ├── 지역별 실거래가 (지도 기반)
  ├── 단지 상세
  │     ├── 동·호수별 거래 내역 (핵심!)
  │     ├── 가격 변동 그래프
  │     ├── 매물 정보
  │     └── 주변 시세 비교
  ├── 분양 정보
  ├── 입주물량 / 미분양
  ├── 학군
  └── AI 가격 예측

## 5. KB부동산

> **"KB부동산 데이터허브"** — 대한민국 부동산 시세의 기준, KB국민은행 운영

### 5-1. 핵심 기능

|------|------|

| **AI추정가(AVM)** | 층수, 향, 면적, 리모델링 여부 등 개별 특성 반영 AI 추정 가격 |
| **KB통계보드** | 핵심 부동산 지표 대시보드 (시장심리, 거래량, 인구변동률 등) |
| **통계기상도** | 부동산 통계를 날씨 뉴스처럼 직관적으로 시각화 |

| **예측시세** | AI 모델 기반 향후 시세 예측 |
| **분양 정보** | 전국 분양 일정 |
| **경매 정보** | 부동산 경매 물건 정보 |
| **대출 연계** | KB국민은행 주담대 연계 |

### 5-2. 정보 계층

```
[KB부동산 홈]
  ├── KB시세 조회 (검색 / 지도)
  │     ├── 단지 시세 (일반시세 + AI추정가)
  │     ├── 실거래가 비교
  │     └── 시세 변동 추이
  ├── KB부동산 데이터허브
  │     ├── KB통계보드 (대시보드)
  │     ├── 통계기상도 (시각화)

  │     └── AI맵 (지역별 AI분석)

### 5-3. 차별화 포인트

- **시세의 표준**: KB시세는 은행 대출 심사, 정부 정책, 감정평가의 사실상 표준. "부동산 가격"이라 하면 KB시세를 의미
- **AI추정가(AVM)**: 약 1,000만 가구 이상의 전국 아파트 동호별 특성을 반영한 AI 모델. 수도권 오차율 4~5%
- **장기 시계열 데이터**: 수십 년간 축적된 시세 데이터와 산출 노하우
- **50가구 미만 소규모 아파트까지**: AI시세를 활용해 기존 KB시세 조사 대상이 아닌 "나홀로 아파트"까지 시세 적용 범위 확대
- **통계기상도**: 복잡한 부동산 통계를 날씨 아이콘으로 직관화 (업계 유일)

**단점:**

- 매물 정보는 약함 (시세 중심이지 거래 중심이 아님)
- 모바일 앱 사용성이 경쟁사 대비 떨어짐
- 일반 사용자에겐 통계가 다소 어려울 수 있음
