### 1.4 릴스 전략

#### 릴스 포맷 3종

**포맷 1: "시세 변동 속보" (15~30초)**
```
[화면 구성]
- 배경: 해당 아파트 단지 로드뷰/항공뷰 (네이버 지도 캡처)
- 상단: 단지명 + 평수 텍스트 오버레이
- 중앙: 가격 카운트다운 애니메이션 (15억 → 9억으로 숫자 떨어지는 효과)
- 하단: "더 많은 폭락 아파트 → 프로필 링크"

[음성/BGM]
- 긴박한 BGM (저작권 무료)
- 내레이션 없음 (자막만, 제작 효율)

[게시 빈도] 주 2회 (화/목)
```

**포맷 2: "1분 주간 부동산 요약" (45~60초)**
```
[화면 구성]
- 슬라이드 전환형 (빠른 컷)
- 1컷: "이번 주 부동산, 1분 요약" 타이틀
- 2컷: 가장 많이 떨어진 아파트 (3초)
- 3컷: 가장 많이 오른 아파트 (3초)
- 4컷: 금리 변동 현황 (3초)
- 5컷: 거래량 핫스팟 (3초)
- 6컷: CTA "매일 업데이트 → @donjup.official 팔로우"

[게시 빈도] 주 1회 (일요일 오후 7시)
```

**포맷 3: "충격 데이터 한 줄" (7~15초)**
```
[화면 구성]
- 검은 배경 + 큰 흰색 텍스트만
- "강남 84㎡ 아파트가 5억 빠졌다고?" (2초)
- 데이터 카드 등장 (3초)
- "나머지 순위는? → 프로필 링크" (2초)

[게시 빈도] 주 2회 (소재가 자극적일 때 수시)
```

#### 릴스 제작 효율화
- **도구**: CapCut 무료 버전 (자막 자동 생성 + 템플릿 저장)
- **제작 시간 목표**: 1개당 15분 이내 (템플릿화 후)
- **소재 수집**: 매일 데일리 리포트에서 가장 자극적인 수치 1개 선별

### 1.6 해시태그 전략 (대형/중형/소형 믹스)

인스타그램 해시태그는 **대형(100K+), 중형(10K~100K), 소형(1K~10K)** 3단계로 믹스해야 탐색 탭 노출 확률이 높아진다.

**게시물당 해시태그 구성 (총 20~25개):**

| 크기 | 비중 | 역할 | 예시 |
|------|------|------|------|
| **대형** (100K+) | 5~7개 | 도달 확장 (경쟁 치열) | #부동산 #아파트 #부동산투자 #재테크 #부동산시장 #아파트매매 #부동산정보 |
| **중형** (10K~100K) | 8~10개 | 탐색 탭 진입 핵심 | #아파트실거래가 #아파트시세 #주담대 #부동산하락 #아파트폭락 #대출금리 #부동산분석 #아파트투자 #부동산데이터 #금리비교 |
| **소형** (1K~10K) | 5~7개 | 상위 노출 + 니치 커뮤니티 | #오늘의부동산 #아파트가격변동 #실거래가분석 #코픽스금리 #부동산숫자 #돈줍 #부동산자동화 |
| **브랜드** (자체) | 2~3개 | 브랜드 검색 + 축적 | #돈줍 #돈줍데일리 #donjup.official |

**콘텐츠별 변동 해시태그:**
```

# 금리 콘텐츠 추가:
#기준금리 #코픽스 #주택담보대출 #대출이자 #금리인하 #금리인상 #대출갈아타기

# 지역 콘텐츠 추가:
```

**해시태그 관리 규칙:**
- 매월 1회 해시태그별 게시물 수 체크 → 포화/신규 태그 교체

- 해시태그를 첫 번째 댓글에 배치 (본문 깔끔하게 유지)

### 2.2 롱테일 키워드 자동 생성 전략

돈줍의 핵심 SEO 무기는 **데이터 기반 페이지 자동 생성**이다. 매일 수집되는 데이터가 곧 롱테일 키워드 페이지가 된다.

**자동 생성 패턴:**
```
/apt/{구}/{단지슬러그}
  → "래미안퍼스티지 실거래가" (단지명 검색)
  → title: "래미안퍼스티지 84㎡ 실거래가 - 최고가 대비 -25% | 돈줍"

/market/{구}
  → "강남구 아파트 시세" (지역 검색)
  → title: "강남구 아파트 실거래가 동향 - 이번 주 거래량 TOP 단지 | 돈줍"

/daily/{날짜}
  → "2026년 3월 22일 부동산 시세" (날짜별 검색)
  → title: "2026.03.22 부동산 데일리 - 폭락 TOP 5, 신고가 경신 3건 | 돈줍"

/rate
  → "주담대 금리" "코픽스 금리"
  → title: "오늘의 주담대 금리 현황 - 기준금리 3.50%, COFIX 3.42% | 돈줍"
```

**핵심:** 각 페이지의 `<title>`과 `<meta description>`에 검색자가 원하는 핵심 숫자(가격, 금리, 변동률)를 포함시켜야 CTR이 올라간다.

### 2.3 네이버 SEO vs 구글 SEO 양면 전략

한국에서 부동산 검색은 네이버가 70%+ 점유. 구글과 네이버는 SEO 전략이 완전히 다르다.

#### 네이버 SEO 전략

| 항목 | 전략 |
|------|------|
| **네이버 서치어드바이저 등록** | 사이트 등록 + 사이트맵 제출 필수 (searchadvisor.naver.com) |
| **네이버 웹마스터 가이드 준수** | meta 태그에 `naver-site-verification` 추가 |

| **키워드 배치** | `<title>` 앞부분에 핵심 키워드 배치 (네이버는 title 앞부분 가중치 높음) |

|------|------|
