## 1. 인스타그램 카드뉴스 전략

### 1.1 채널 개요

| 항목 | 내용 |
|------|------|
| 계정명 | @donjupkr |
| 포지셔닝 | "매일 아침, 부동산 숫자로 말하는 채널" |
| 톤앤매너 | 자극적 숫자 + 심플한 디자인 + 공포/FOMO 자극 |
| 주 타겟 | 30~40대 부동산 관심층 (인스타 활성 사용자) |
| 핵심 KPI | 저장수 > 공유수 > 좋아요수 (알고리즘 가중치 순) |

#### 시리즈 B: "금리 한 입" (주 2회, 수/금)

| 항목 | 내용 |
|------|------|
| 형식 | 카드뉴스 3장 |
| 게시 시간 | 수/금 오후 12:30 (점심시간) |
| 데이터 소스 | finance_rates 테이블 |

**카드 구성**:
```

      기준금리 3.50% (동결) / COFIX 3.42% (▼0.02%p)

      원리금균등 134만원 / 원금균등 125만원 비교

[3장] "내 대출이자 직접 계산해보기 → 돈줍 금리계산기"
```

## 2. SEO 키워드 전략

### 2.1 키워드 매트릭스

| 카테고리 | 타겟 키워드 예시 | 대응 페이지 | 검색 의도 | 예상 경쟁도 |
|---------|----------------|-----------|----------|-----------|
| **단지명** | "래미안퍼스티지 실거래가", "헬리오시티 시세" | `/apt/[region]/[apt-id]` | 정보형 | 중 |
| **지역** | "강남구 아파트 시세", "마포구 부동산 동향" | `/market/[region]` | 정보형 | 높음 |
| **금리** (고단가) | "주담대 금리 비교", "대출이자 계산기" | `/rate/calculator` | 거래형 | 높음 |
| **시사** | "2026년 3월 부동산 시장", "오늘 아파트 폭락" | `/daily/[date]` | 정보형 | 낮음 |
| **롱테일** | "XX아파트 30평 실거래가 2026", "COFIX 금리 변동" | 상세 페이지들 | 정보형 | 매우 낮음 |

### 2.2 SEO 기술 전략

#### 메타태그 동적 생성

```typescript
// app/apt/[region]/[aptId]/page.tsx
export async function generateMetadata({ params }): Promise<Metadata> {
    const apt = await getAptDetail(params.aptId);
    const latest = apt.latestTransaction;

    return {
        title: `${apt.apt_name} 실거래가 - 돈줍`,
        description: `${apt.apt_name} ${latest.size_sqm}㎡ 최근 거래가 ${formatPrice(latest.trade_price)}. 최고가 대비 ${latest.change_rate}%. 전체 거래 내역과 가격 추이를 확인하세요.`,
        openGraph: {
            title: `${apt.apt_name} 실거래가`,
            description: `최고가 대비 ${latest.change_rate}% | 최근 거래 ${formatPrice(latest.trade_price)}`,
            images: [`/api/og/apt/${params.aptId}`],  // 동적 OG 이미지
        },
    };
}
```

#### JSON-LD 구조화 데이터

각 아파트 상세 페이지에 `Product` 스키마 적용하여 검색 결과에 가격 정보 노출:

```json
{
    "@context": "https://schema.org",
    "@type": "Product",
    "name": "래미안퍼스티지 84㎡",
    "description": "서울시 강남구 대치동 래미안퍼스티지 아파트 실거래가 정보",
    "offers": {
        "@type": "Offer",
        "price": "970000000",
        "priceCurrency": "KRW",
        "availability": "https://schema.org/InStock"
    }
}
```

#### 사이트맵 자동 생성

```typescript
// app/sitemap.ts
export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
    const complexes = await getAllComplexSlugs();
    const reports = await getAllReportDates();

    return [
        { url: 'https://donjup.com', changeFrequency: 'daily', priority: 1.0 },
        { url: 'https://donjup.com/rate/calculator', changeFrequency: 'monthly', priority: 0.9 },
        ...complexes.map(c => ({
            url: `https://donjup.com/apt/${c.region}/${c.slug}`,
            changeFrequency: 'weekly' as const,
            priority: 0.8,
        })),
        ...reports.map(r => ({
            url: `https://donjup.com/daily/${r.date}`,
            changeFrequency: 'never' as const,
            priority: 0.6,
        })),
    ];
}
```

### 2.3 페이지 색인 성장 전략

| 기간 | 예상 색인 페이지 수 | 구성 |
|------|-------------------|------|
| 런칭 시 | ~100개 | 메인 + 주요 단지 50개 + 지역 25개 + 기타 |
| 1개월 후 | ~500개 | + 단지 확대 + 데일리 리포트 30개 |
| 3개월 후 | ~2,000개 | + 전국 주요 단지 + 리포트 90개 |
| 6개월 후 | ~5,000개 | + 전국 확대 + 리포트 180개 |
| 1년 후 | ~10,000개+ | 전국 단지 + 리포트 365개 + 지역 페이지 |

> 핵심: 데일리 리포트가 매일 1개씩 자동 생성되므로 SEO 롱테일 페이지가 자동 축적된다.

## 4. 광고 배치 전략

### 4.1 애드센스 배치 원칙

| 원칙 | 설명 |
|------|------|
| 모바일 우선 | 트래픽의 70%+ 모바일 예상. 모바일 레이아웃 기준으로 광고 위치 설계 |
| 페이지당 3~4개 | 과도한 광고는 UX 훼손 → 이탈률 증가 → 장기 수익 감소 |
| 수동 배치 | 자동 광고(Auto Ads) OFF → 수동으로 최적 위치에 배치 |
| 앵커 광고 활용 | 모바일 하단 고정 앵커 광고 전 페이지 활성화 |
| 콘텐츠 연관성 | 금리 페이지에 대출 광고, 아파트 페이지에 부동산 광고 배치 |

│ │ [토스 대출 갈아타기 →]       │ │

### 4.3 CPA 제휴 전략

| 제휴 파트너 | 연결 페이지 | CTA 문구 | 예상 건당 수수료 |
|------------|-----------|---------|----------------|
| 뱅크샐러드 | 금리 계산기 | "최저 금리 한번에 비교하기" | 3,000~5,000원 |
| 핀다 | 금리 계산기 | "나에게 맞는 대출 찾기" | 5,000~10,000원 |
| 토스 | 금리 계산기 + 아파트 상세 | "대출 갈아타기 시뮬레이션" | 3,000~8,000원 |
| 카카오뱅크 | 금리 대시보드 | "비대면 주담대 금리 확인" | 5,000~15,000원 |

**배치 원칙**:
- 계산 결과 직후가 CTR 최고 순간 → 여기에 가장 높은 수수료 파트너 배치
- 자연스러운 맥락에서 노출 (하드셀 X, "더 나은 조건 찾기" 느낌으로)
- 각 링크에 UTM 파라미터 부착: `?utm_source=donjup&utm_medium=cpa&utm_campaign=calculator`
- A/B 테스트: 2주 단위로 CTA 문구/위치 변경하여 전환율 비교

## 5. 성장 전략

### 5.1 초기 성장 (0~3개월)

| 전략 | 실행 |
