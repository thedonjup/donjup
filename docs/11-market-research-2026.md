## 1. 한국 경쟁사 심층 분석

### 1-1. 호갱노노 (hogangnono.com)

> **"아파트 실거래가 1등 앱"** — 직방 자회사, 200만+ 회원

#### 2026년 최신 기능

| 기능 | 상세 설명 | 돈줍 대비 차별점 |
|------|----------|----------------|
| **3D 단지투어** | 방문 없이 3D로 아파트 단지 전경, 세대별 조망, 시간대/계절별 일조량 확인. 봄/여름/가을/겨울 오전 8시~오후 6시 시뮬레이션 | 돈줍에 없음 (기술적 진입장벽 높음) |
| **실시간 랭킹** | 청약, 재건축, 동네 이슈 등 인기 단지/지역 실시간 업데이트 | 돈줍은 일일 업데이트 수준 |
| **고급 필터** | 매매/전세/월세, 평형, 세대수, 금액, 입주년차, 용적률, 갭가격, 월세수익률, 전세가율, 주차공간 등 | 돈줍은 기본 필터만 제공 |
| **부동산 뉴스** | 시세 영향 주요 뉴스, 지역별 뉴스 매일 업데이트 | 돈줍에 없음 |
| **등기 알림** | 소유권 변동 시 알림 서비스 | 돈줍에 없음 |

#### 수익 모델
- **공인중개사 광고**: 노출 비율에 따른 광고비 (스페셜 AD / 프리미엄 AD / 기본 AD)
- **분양 광고**: 신규 분양 아파트 홍보
- **성과형 과금**: 노출된 만큼만 비용 부과, 효과 없으면 중단 가능

#### 사용자 불만 사항
- 유료 기능과 무료 기능의 경계가 모호
- 지방 아파트 데이터 부족
- 앱 용량 과대 (3D 기능 때문)

#### 돈줍이 배울 점
- **실시간 랭킹 시스템**: 돈줍의 일일 리포트를 실시간으로 업그레이드 가능
- **다양한 필터**: 갭가격, 전세가율, 월세수익률 등 투자 관점 필터 추가 필요
- **뉴스 큐레이션**: 시세에 영향을 주는 뉴스 자동 연동 고려

### 1-2. 아실 (asil.kr)

> **"아파트는 아실"** — 280만+ 다운로드, 평점 4.6

#### 2026년 최신 기능

| 기능 | 상세 설명 | 돈줍 대비 차별점 |
|------|----------|----------------|
| **매수심리 지수** | 지역별 매수/매도 우위 지수 시각화 | 돈줍에 없음 — 시장 심리 데이터 부재 |
| **학군 비교** | 학업 성취도, 특목고 진학률 기반 학교 랭킹 | 돈줍에 없음 — 핵심 차별화 포인트 |
| **단지 비교 그래프** | 여러 단지 가격 변동을 하나의 그래프로 비교 | 돈줍에 없음 — 매우 유용한 기능 |
| **갭투자 분석** | 매매가-전세가 차이 분석 | 돈줍에 없음 |
| **거래량 분석** | 지역별/단지별 거래량 추이 | 돈줍은 기본 수준만 제공 |
| **외지인 투자 커뮤니티** | 비거주 투자자 전용 커뮤니티 | 돈줍에 없음 |
| **전문가 영상 2,600건+** | 부동산 전문가 영상 큐레이션 | 돈줍에 없음 |

#### 고유 강점 (USP)
- **공공데이터 활용 우수**: 국토교통부, 한국부동산원 API를 적극 활용
- **투자자 관점 최적화**: 갭투자, 매수심리, 외지인 거래 등 투자 분석 특화
- **무료 기반 확장**: 기본 기능 무료, 프리미엄은 일부 고급 분석만

#### 사용자 참여 전략
- 유튜브 연동 전문가 콘텐츠
- 커뮤니티를 통한 사용자 잔존율 확보
- 분양 정보 푸시 알림

#### 돈줍이 배울 점
- **매수심리 지수 도입**: KB부동산 데이터 또는 자체 계산 지표로 구현 가능
- **학군 정보 연동**: 교육부 공공데이터 활용 (학업성취도, 진학률)
- **단지 비교 기능**: 돈줍의 차트 인프라(Recharts)로 구현 용이

### 1-3. 직방 (zigbang.com) / 다방 (dabang.com)

> **직방**: 연 매출 1,014억원 (2024), 프롭테크 최대 기업
> **다방**: 스테이션3 운영, 연 매출 246억원

#### 2026년 AI 핵심 기능

| 플랫폼 | AI 기능 | 상세 |
|--------|---------|------|
| **직방** | AI 중개사 | 자연어 대화로 매물 탐색. "관리비 저렴하고 주차 가능한 2룸" 같은 일상 표현으로 검색 |
| **직방** | 조건 확장 AI | 매칭 매물 없을 시 AI가 단계적으로 조건 완화하여 대안 제시 |
| **직방** | 다국어 지원 | 영어/일본어/중국어 질문 시 해당 언어로 답변 |
| **다방** | AI 상세설명 자동 생성 | 매물 등록 시 AI가 정보 분석하여 상세설명 자동 작성 |

#### 멀티 물건 유형 처리
- **직방**: 아파트, 원투룸, 오피스텔, 빌라, 상가 등 전 주거 유형 커버
- **다방**: 원투룸/오피스텔 특화, 아파트로 확장 중

#### 지도 UX 패턴
- 줌 레벨별 클러스터링 (시도 > 시군구 > 단지)
- 지도 위 가격 버블 표시 (색상으로 상승/하락 구분)
- 로드뷰 연동 (주변 환경 확인)
- 통근 시간 오버레이

#### 수익 구조
- **중개사 광고**: 매물 노출 광고 (주 수익원, 전체의 ~65%)
- **분양 광고**: 신규 분양 단지 프로모션
- **스마트홈 하드웨어**: 직방 자체 스마트홈 기기 매출 (수익 다각화)
- **프리미엄 매물**: 상위 노출 유료 옵션

#### 돈줍이 배울 점
- **자연어 검색**: 기술적으로 OpenAI API 연동으로 구현 가능 (비용 고려 필요)
- **지도 UX**: 줌 레벨별 데이터 밀도 조절은 필수
- **다국어**: 외국인 투자자/거주자 시장 고려

### 1-4. 부동산지인 (aptgin.com)

> **"부동산 빅데이터 플랫폼"** — 전문 투자자 타겟

#### 2026년 핵심 분석 기능

| 기능 | 상세 | 활용 대상 |
|------|------|----------|
| **빅데이터 지도** | 시세, 입주물량, 전출입, 거래량을 지도 위에 시각화 | 전문 투자자 |
| **지역 분석** | 상승/하락 가능성 데이터 기반 탐색 | 지역 선정 |
| **시장 분석** | 매도/매수 시점 판단, 향후 가격 증감 추정 | 타이밍 분석 |
| **가격 분위별 분석** | 저가/중가/고가 아파트별 거래 동향 | 투자 전략 수립 |
| **AI 자체 시세** | AI 기반 시세 생성 및 제공 | 정밀 가격 분석 |
| **원클릭 리포트** | 관심 지역/아파트 선택 시 종합 리포트 자동 생성 | 의사결정 지원 |
| **전출입 분석** | 인구/세대의 지역 간 이동 현황 | 수요 예측 |
| **입주 물량 분석** | 지역별/기간별 수요-입주량 및 미분양 | 공급 분석 |

#### 수익 모델
- **프리미엄 구독**: 60개+ 분석 페이지 유료 제공
- **기업 회원 이용권**: B2B 대상 별도 요금제
- **기본 무료 + 고급 유료**: 프리미엄 전환 유도

#### 전문 투자자가 사용하는 이유
1. 데이터 시각화 품질이 높음
2. 입주물량-가격 상관관계 분석 가능
3. 전출입 데이터로 수요 예측
4. AI 시세가 실거래가와 별도로 제공되어 비교 분석 가능

#### 돈줍이 배울 점
- **입주 예정 물량**: 공공데이터 활용하여 지역별 입주 예정 데이터 연동
- **전출입 분석**: 통계청 API 활용 가능
- **원클릭 리포트**: 돈줍의 일일 리포트를 단지별로 확장 가능
- **프리미엄 구독 모델**: 수익화 참고 사례

## 2. 글로벌 프롭테크 트렌드 2026

### 2-1. AI 트렌드

#### Agentic AI (에이전틱 AI) — 2026 최대 키워드

> 단순 콘텐츠 생성을 넘어 **자율적 의사결정과 실행**을 하는 AI

- **Zillow**: ChatGPT 앱 출시, Zestimate에 기후 위험 스코어링(산불/홍수) 통합
- **Redfin**: ChatGPT 앱 + "Ask Redfin" 대화형 검색 (다국어 지원)
- **직방**: AI 중개사 전 주거유형 확대, 자연어 매물 탐색

| 적용 영역 | 글로벌 트렌드 | 돈줍 적용 가능성 |
|----------|-------------|----------------|
| 대화형 매물 검색 | Zillow, Redfin, 직방 모두 도입 | **중** — OpenAI API 비용 부담 |
| AI 시세 예측 | KB부동산, 부동산R114, 부동산플래닛 | **상** — 실거래가 데이터 보유 |

| AI 매물 설명 생성 | 다방 AI 상세설명 | **하** — 매물 서비스 아님 |

#### Spatial AI (공간 AI) — 차세대 기술

- 언어 모델을 넘어 물리적 공간을 이해하는 AI
- 건설 현장에서 스마트 글라스로 실시간 공간 데이터 제공
- 부동산에서는 3D 모델링 + AI 분석 결합

## 핵심 요약 및 액션 아이템

### 돈줍의 현재 경쟁 포지션
- **강점**: 일일 리포트 자동 생성, 대출 금리/계산기, 카드뉴스/SNS 자동 시딩 — 이 세 가지는 경쟁사에 없는 독보적 기능
- **약점**: 데이터 분석 깊이, 학군/교통/입주물량 등 부가 정보 부재, 커뮤니티 부재
- **기회**: 공공데이터 API가 풍부하게 개방되어 있어 적은 비용으로 데이터 확장 가능
- **위협**: 네이버, 직방 등 대형 플랫폼의 AI 투자 가속화

### 즉시 실행해야 할 TOP 5 액션

1. **청약 정보 연동** — 한국부동산원 API 연동, 청약 캘린더 페이지 추가
2. **학군 정보 추가** — 학교알리미 API 연동, 단지 상세 페이지에 학군 탭 추가
3. **단지 비교 기능** — 2~4개 단지 선택 비교 (가격 차트 + 기본 정보)
4. **투자 수익률 시뮬레이터** — 기존 대출 계산기 확장
5. **대출 제휴 CPA** — 토스/뱅크샐러드 제휴 링크로 즉시 수익 확대
