# 돈줍(DonJup) 성장 전략서 — 실행 매뉴얼

> **목표**: 1인 운영, 자동화 85%+, 주당 관리 5~6시간으로 월 100만원+ 수익 달성
> **원칙**: "이게 돈이 되나?" — 모든 기능/전략은 수익 연결이 명확해야 함
> **작성일**: 2026-03-22

## 1. 수익 최단 경로 — 애드센스 승인 전에 돈 버는 법

### 1.1 Day 1부터 가능한 수익화 채널

애드센스는 승인까지 1~4주 걸린다. 그 전에 바로 수익을 만드는 채널부터 세팅한다.

| 수익원 | 가입 즉시 | 승인 소요 | 예상 수익 | 돈줍 적용 |
|--------|----------|----------|----------|----------|
| **쿠팡파트너스** | O (즉시) | 없음 | 구매액 3~3.5% | 부동산 관련 도서/인테리어 상품 배너 |
| **텐핑** | O (즉시) | 없음 | 건당 2,000~10,000원 (금융) | 금융 앱 CPA (대출비교, 신용조회 등 고단가) |
| **카카오 애드핏** | 심사 3~5일 | 짧음 | RPM 1,000~3,000원 | 애드센스 대안/보완 |
| **핀다/뱅크샐러드 CPA** | 제휴 신청 | 1~2주 | 건당 5,000~15,000원 | 금리 계산기 결과 직후 |
| **구글 애드센스** | 심사 필요 | 1~4주 | RPM 4,000~8,000원 | 전 페이지 메인 광고 |

### 1.2 수익화 세팅 순서 (첫 2주)

```
Day 1:  쿠팡파트너스 가입 → 부동산 도서 배너 사이드바 배치
Day 1:  텐핑 가입 → 금융 앱 CPA 배너 세팅
Day 2:  카카오 애드핏 신청
Day 3:  핀다/뱅크샐러드 제휴 신청
Day 3:  구글 애드센스 신청
Day 5:  카카오 애드핏 승인 → 광고 슬롯 배치
Week 2: 핀다/뱅크샐러드 CPA 승인 → 금리 계산기 연동
Week 3~4: 애드센스 승인 → 전면 광고 세팅
```

**핵심**: Day 1부터 쿠팡파트너스 + 텐핑으로 미미하지만 수익이 발생한다. 이후 CPA → 애드핏 → 애드센스 순으로 수익 레이어를 쌓는다.

### 1.3 자동화된 수익 트리거

사용자 행동에 따라 자동으로 수익화 포인트가 노출되는 구조:

```
[사용자 행동]                    → [자동 수익 트리거]
─────────────────────────────────────────────────────
폭락 아파트 카드 클릭             → 단지 상세 진입 → 인피드 광고 노출
단지 상세에서 가격 확인           → "이 가격에 대출받으면?" CTA 자동 노출
금리 계산기 결과 확인             → CPA 배너 3단계 자동 노출 (핀다/뱅크샐러드/신용점수)
계산기에서 금리 0.5%p 절약액 표시  → "더 낮은 금리 찾기" 감정 자극 → CPA 클릭
데일리 리포트 열람                → 중간 인피드 광고 + 하단 배너
사이드바 체류                    → 쿠팡파트너스 부동산 도서 배너
```

모든 트리거는 코드로 구현되므로 한 번 세팅하면 사람 손 없이 영구 작동한다.

## 2. 트래픽 유입 전략

### 2.0 핵심 원칙: 초기 트래픽 부트스트랩

첫 1개월에 가장 중요한 것은 **인스타 팔로워 수가 아니라 웹사이트 유입**이다. 팔로워 0에서 인스타만으로는 하루 10~20명이 한계. 따라서 **커뮤니티 시딩 + 카카오 오픈채팅 + 네이버 블로그를 Day 1부터 병행**해야 한다.

```
초기 1개월 트래픽 소스 (현실적):
├── 커뮤니티 시딩 (디시/뽐뿌/에펨코리아): 40% — 즉시 유입, 하루 30~100명
├── 카카오 오픈채팅 부동산방 링크 공유: 20% — 즉시 유입, 하루 20~50명
├── 네이버 블로그: 20% — 1~2주 후 검색 유입 시작
├── 인스타그램: 15% — 팔로워 쌓이면서 점진적 증가
└── SEO: 5% — 3개월 후부터 본격
```

### 2.1 채널별 우선순위 (수정)

| 순위 | 채널 | 시작 시점 | 역할 | 자동화율 | 예상 일 유입 |
|------|------|----------|------|---------|------------|
| 1 | **커뮤니티 시딩** | **Day 1** | 즉시 트래픽 | 50% | 30~100명 |
| 2 | **카카오 오픈채팅** | **Day 1** | 즉시 트래픽 + 재방문 | 60% | 20~50명 |
| 3 | **인스타그램 카드뉴스** | Week 1 | 브랜드 + 점진적 성장 | 90% | 10~50명 (초기) |
| 4 | **네이버 블로그** | **Week 1** | 네이버 검색 유입 (한국 70%) | 70% | 20~100명 (2주 후) |
| 5 | **인스타 릴스** | Week 3 | 도달 확장 (릴스 도달 2배) | 85% | 추가 도달 |
| 6 | **SEO (자동 페이지)** | Week 1 | 복리 트래픽 엔진 | 100% | 3개월 후 폭발 |
| 7 | **유튜브 쇼츠/틱톡** | Week 7+ | 스케일업 | 85% | 추가 채널 |

### 2.1.1 신규: 카카오 오픈채팅 전략

부동산 관련 카카오 오픈채팅방은 수천~수만명이 참여하고 있다. 여기에 매일 데이터를 공유하면 즉시 유입된다.

**실행 방법:**
1. "부동산 투자", "아파트 실거래가", "내 집 마련" 등 키워드로 오픈채팅방 10개+ 참여
2. 매일 아침 자동 생성된 텍스트를 공유:
   > "오늘 서울 아파트 폭락 TOP 3
   > 1위: XX아파트 -35% (9.7억→6.3억)
   > 2위: OO아파트 -28%
   > 전체 순위: donjup.com"
3. 텍스트는 Cron으로 자동 생성 → 카톡에 복붙만 수동 (1분)

**주의:** 도배하면 추방됨. 하루 1회, "정보 공유자" 포지션 유지.
**예상 효과:** 방 10개 × 클릭 5~10명 = 하루 50~100명 유입

### 2.1.2 커뮤니티 시딩 강화 (Week 4 → Day 1로 앞당김)

커뮤니티 시딩은 초기 가장 빠르게 트래픽을 만드는 채널이므로 **Day 1부터 시작**한다.

| 커뮤니티 | 난이도 | 시작 | 방법 | 예상 유입/글 |
|---------|--------|------|------|------------|
| **에펨코리아** | 쉬움 | Day 1 | "와 XX아파트 8억→4.6억 ㄷㄷ [출처: 돈줍]" | 30~100명 |
| **뽐뿌** | 쉬움 | Day 1 | 자유게시판 데이터 공유 | 20~80명 |
| **디시 부동산갤** | 보통 | Day 3 | 자극적 데이터 + 출처 링크 | 50~200명 |
| **클리앙** | 보통 | Week 2 | IT인 대상, 데이터 분석 톤 | 30~100명 |
| **네이버카페 부동산스터디** | 어려움 | Week 3 | 가입+등급 필요, 장기 투자 | 100~300명 |
| **월급쟁이부자들 카페** | 어려움 | Week 3 | 등급제, 고품질 글 필요 | 100~300명 |

**시딩 텍스트 자동 생성** (매일 Cron):
```
Cron → DB 쿼리 → 커뮤니티별 톤 맞춤 텍스트 3종 생성
  ├── 디시/에펨용: 반말, 짧게, 자극적
  ├── 네이버카페용: 존댓말, 상세, 분석적
  └── 클리앙용: 팩트 위주, 데이터 중심
→ Supabase seeding_queue 저장 → 이메일/슬랙 알림 → 수동 복붙 게시
```

**주 투입 시간:** 주 1시간 (하루 10분 × 6일)

### 2.1.3 바이럴 트리거 — 공유/저장 유도 장치

콘텐츠를 "보기만"하는 게 아니라 **공유하게 만드는** 장치가 필요하다.

**카드뉴스 마지막 장 CTA 변형 (A/B 로테이션):**
- A: "이 동네 사는 친구한테 보내줘" → **태그 유도**
- B: "내가 보는 아파트는 얼마나 빠졌을까? → donjup.com" → **웹 유입**
- C: "저장해두면 나중에 비교하기 좋아요 💾" → **저장 유도**
- D: "주변에 영끌한 친구 있으면 공유 🔥" → **공유 유도**

**릴스 후킹 강화:**
- 첫 1초: 화면 가득 빨간 숫자 "-42%" + 효과음 → 스크롤 멈춤
- 마지막 3초: "팔로우하면 매일 아침 폭락 알림" → 팔로우 유도

### 2.2 Phase 1: 인스타그램 집중 (Week 1~6)

#### 카드뉴스 — 매일 자동 발행 (자동화 90%)

**콘텐츠 3종 로테이션:**

| 시리즈 | 빈도 | 게시 시간 | 수익 연결 |
|--------|------|----------|----------|
| "오늘의 폭락/신고가" | 매일 | 08:30 | 카드 5장 중 마지막 → "더 보기 → donjup.com" → 웹 유입 → 광고 |
| "금리 한 입" | 주 2회 (수/금) | 12:30 | "내 대출이자 직접 계산" → 금리 계산기 → CPA |
| "이 동네 무슨 일?" | 주 1회 (토) | 10:00 | "실시간 거래 동향" → 지역 페이지 → 광고 |

**자동화 파이프라인:**
```
[매일 08:00] Supabase 데이터 조회 (폭락 TOP 3, 신고가 TOP 3)
     ↓
[자동] Vercel API Route → @vercel/og (Satori) 기반 카드뉴스 이미지 5장 생성
     ↓
[자동] Instagram Graph API로 자동 게시 (Business 계정 필수)
     ↓
[자동] 해시태그 20~25개 첫 번째 댓글로 자동 추가
```

**기술 스택 (비용 0원):**
- 이미지 생성: `@vercel/og` (Satori + Resvg) — Next.js API Route에서 JSX → PNG 변환. Vercel 무료 플랜으로 충분
- 자동 게시: Instagram Graph API (Business/Creator 계정, Facebook App 필요)
- 스케줄링: Vercel Cron Jobs (무료 플랜: 1일 1회) 또는 Vercel Cron Jobs (무료)
- 폰트: Pretendard (무료, self-hosted)

**수동 작업 (주당 30분):**
- 주 1회 인사이트 확인 → 저성과 해시태그 교체
- 댓글/DM 확인 (일 5분)

#### 릴스 — 슬라이드쇼 방식 (자동화 85%)

복잡한 영상 편집 없이, 카드뉴스 이미지를 슬라이드쇼 영상으로 변환한다.

**방법:**
```
카드뉴스 이미지 5장 (이미 자동 생성됨)
     ↓
FFmpeg로 슬라이드쇼 영상 생성 (각 이미지 3초 × 5장 = 15초)
     ↓
저작권 무료 BGM 자동 합성 (긴박한 분위기, 미리 5~10곡 준비)
     ↓
릴스로 자동 업로드 (Instagram Graph API - Reels 지원)
```

**FFmpeg 명령 (Vercel Cron Jobs에서 실행):**
```bash
ffmpeg -framerate 1/3 -i card_%d.png -i bgm.mp3 \
  -c:v libx264 -r 30 -pix_fmt yuv420p \
  -shortest -y output.mp4
```

**빈도:** 주 2~3회 (화/목 + 자극적 데이터 있을 때)
**수동 작업:** BGM 교체 월 1회 (10분)

### 2.3 Phase 2: 멀티 플랫폼 확장 (Week 7~12)

인스타에서 검증된 콘텐츠를 다른 플랫폼으로 확장한다.

| 플랫폼 | 콘텐츠 | 추가 작업 | 자동화 |
|--------|--------|----------|--------|
| **유튜브 쇼츠** | 릴스 영상 그대로 | YouTube Data API v3 업로드 | 90% |
| **틱톡** | 릴스 영상 그대로 | TikTok API 업로드 | 90% |
| **네이버 블로그** | 주간 리포트 텍스트 | 주 2회, 마크다운 → HTML 변환 | 70% |

**핵심: 하나의 데이터 → 5개 플랫폼 자동 배포**
```
Supabase 데이터
  ├── → 카드뉴스 이미지 → 인스타 피드
  ├── → 슬라이드쇼 영상 → 인스타 릴스 + 유튜브 쇼츠 + 틱톡
  ├── → 데일리 리포트 웹 → SEO 인덱싱
  └── → 주간 요약 텍스트 → 네이버 블로그
```

### 2.4 네이버 블로그 전략 강화 (Week 1부터)

한국 검색 점유율 70%가 네이버. 구글 SEO보다 네이버 블로그가 한국에서 **훨씬 빠르게** 트래픽을 만든다.

**블로그 운영 방식:**
- 블로그명: "돈줍 - 매일 업데이트 부동산 데이터"
- 매일 데일리 리포트 초안을 자동 생성 → 수동으로 블로그에 편집 게시
- 핵심: 모든 글에 "상세 데이터는 → donjup.com" 링크 삽입

**게시 전략:**
| 빈도 | 콘텐츠 | 타겟 키워드 | 예상 검색 유입 |
|------|--------|-----------|-------------|
| 매일 | "2026년 3월 22일 서울 아파트 폭락 순위" | "아파트 폭락", "서울 아파트 하락" | 20~100/일 |
| 주 2회 | "이번 주 금리 변동 + 대출이자 시뮬레이션" | "주담대 금리", "대출이자 계산" | 30~150/일 |
| 주 1회 | "XX구 아파트 거래량 급증, 무슨 일?" | "XX구 아파트", "부동산 동향" | 10~50/일 |

**자동화:** 블로그 글 초안 자동 생성 (Cron → 마크다운 → 이메일 전송) → 수동 편집+게시 (10분/글)
**주당 소요:** 30~40분

### 2.5 SEO — 복리 트래픽 엔진 (자동화 100%)

SEO는 한 번 세팅하면 사람 손 없이 매일 콘텐츠가 축적된다.

**자동 생성 페이지:**
- 단지 상세 페이지: 수천 개 (데이터 있는 만큼 자동)
- 데일리 리포트: 연 365개 (매일 자동)
- 지역 동향 페이지: 서울 25개 구 (자동 갱신)

**SEO 체크리스트 (1회 세팅):**
- [ ] Google Search Console + 사이트맵 자동 갱신
- [ ] 네이버 서치어드바이저 등록
- [ ] 각 페이지 `<title>`에 핵심 숫자 포함 (예: "래미안퍼스티지 84㎡ 실거래가 9.7억 | 돈줍")
- [ ] JSON-LD 구조화 데이터 (BreadcrumbList, FAQ)
- [ ] OG 이미지 자동 생성 (공유 시 미리보기)

**SEO 효과 타임라인:**
- Month 1~2: 인덱싱 시작, 거의 0 유입

### 4.3 CPA 수익이 핵심인 이유

```
애드센스 PV 2,000건 수익 = 약 10,000원
CPA 1건 전환 수익        = 약 10,000원
```

**결론:** CPA 1건 = 애드센스 2,000 PV. 금리 계산기 → CPA 전환 동선 최적화가 수익의 핵심.

**전환 동선 최적화 포인트:**
1. 메인 대시보드 폭락 카드 → 단지 상세 → "이 가격에 대출받으면?" CTA → 계산기 → CPA
2. 인스타 "금리 한 입" 카드뉴스 → 프로필 링크 → 금리 계산기 → CPA
3. 단지 상세 사이드바 → "OO아파트 X억 대출 시 월 이자는?" → 계산기 → CPA
4. SEO "3억 대출 이자" 검색 유입 → 금리 계산기 직행 → CPA
