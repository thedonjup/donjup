## 1. 서비스 개요

### 1.1 서비스 정의

| 항목 | 내용 |
|------|------|
| 서비스명 | **돈줍(DonJup)** |
| 도메인 | donjup.com |
| 핵심 가치 | 매일 자동 업데이트되는 극단적 부동산 가격 변동 + 체감형 금리 정보를 한눈에 |
| 한 줄 소개 | "오늘 가장 많이 떨어진 아파트, 오늘 신고가를 찍은 아파트 — 매일 아침 자동 업데이트" |

### 1.2 타겟 사용자 페르소나

| 페르소나 | 특징 | 핵심 니즈 | 유입 경로 |
|----------|------|----------|----------|
| **A. 30대 실수요자** | 내 집 마련 준비 중, 매일 실거래가 확인 | "내가 보는 단지가 지금 비싼 건지 싼 건지" | 인스타 카드뉴스 → 단지 상세 |
| **B. 40대 투자자** | 지역별 거래량/가격 동향으로 투자 타이밍 판단 | "지금 거래가 몰리는 지역이 어디인지" | 검색(SEO) → 지역 동향 |
| **C. 영끌족** | 주담대 이자 변동에 극도로 민감, 금리 0.1%p도 체크 | "금리가 바뀌면 내 이자가 얼마나 바뀌는지" | 인스타/검색 → 금리 계산기 |

### 1.3 핵심 콘텐츠 원칙

사람들은 **평범한 정보에는 반응하지 않는다.** 오직 두 가지 감정만 자극한다:

- **공포(손실)**: "최고가 15억이 9억으로?" → 폭락 아파트 랭킹
- **FOMO(수익)**: "남들 다 떨어지는데 이 단지만 신고가?" → 신고가 갱신 단지

방대한 공공데이터 중 **[단기 폭락/상승 아파트 리스트] + [금리 변동 지표]** 딱 두 가지만 매일 자동으로 발라내어 가공하는 것이 서비스의 핵심 엔진이다.

### 2.7 돈줍만의 차별점 (경쟁 해자)

경쟁사들은 모두 **"종합 부동산 정보 플랫폼"**을 지향한다. 돈줍은 이 레드오션에서 싸우지 않는다.

#### 핵심 해자 3가지

**1. 감정 자극형 콘텐츠 특화 (Fear & FOMO)**
- 경쟁사: "래미안 84㎡ 실거래가 12억" (정보 나열)
- 돈줍: **"최고가 15억이었던 이 아파트, 오늘 9억에 거래됨 (-40%)"** (감정 자극)
- 같은 데이터를 "극단값 랭킹"으로 재가공 → 인스타 바이럴에 최적화된 포맷
- 경쟁사는 이 방식을 "채택하기 어려움" — 종합 정보 플랫폼의 톤앤매너와 충돌

**2. 서버비 0원 구조 = 손익분기점 즉시 도달**
- 호갱노노: 직방 인프라 의존 (수십억 운영비)
- 아실/리치고: 서버/인프라 비용 매월 수백만~수천만원
- **돈줍: Vercel 무료 + Supabase 무료 + 공공 API = 고정비 0원**
- 월 29만원(1만 PV)부터 순이익. 경쟁사는 수천만 PV가 있어야 BEP

**3. SEO + SNS 바이럴 이중 트래픽 엔진**
- 경쟁사: 앱 다운로드 중심 → 앱스토어 경쟁, 마케팅 비용 필요
- 돈줍: 웹 퍼스트 → **SEO 롱테일**(연 365개 데일리 리포트 + 수만 개 단지 페이지) + **인스타 바이럴**(무료 트래픽)
- 인스타 카드뉴스 → 웹 유입 → SEO 인덱싱 → 검색 유입 → 복리 성장

#### 돈줍이 경쟁사를 대체하지 않아도 되는 이유

돈줍의 사용자는 호갱노노/아실을 **이미 쓰고 있는 사람**이다. 돈줍은 "오늘 뭐가 제일 많이 떨어졌지?"라는 **일일 체크 습관**을 만드는 서비스로, 기존 앱과 경쟁이 아닌 **보완 관계**다. 매일 아침 돈줍 → 관심 단지 발견 → 호갱노노/아실에서 상세 확인. 이 흐름에서 돈줍은 **입구(상류)**를 점유한다.

### 2.8 수익 다각화 로드맵

| Phase | 시기 | 수익원 | 예상 기여 | 설명 |
|-------|------|--------|----------|------|
| **Phase 1** | 0~3개월 | 구글 애드센스 | 100% | MVP 단계. 트래픽 확보에 집중 |
| **Phase 2** | 3~6개월 | + CPA 제휴 (핀다/뱅크샐러드/토스) | 애드센스 60% + CPA 40% | 금리 계산기 완성 후 CPA 연동 |
| **Phase 3** | 6~12개월 | + 프리미엄 구독 (월 9,900원) | 애드센스 40% + CPA 35% + 구독 25% | 지역별 심층 분석, 폭락 알림, AI 예측 |
| **Phase 4** | 12~18개월 | + B2B 데이터 API 라이선스 | 다각화 완료 | 부동산 미디어/유튜버/중개업소에 데이터 API 제공 |
| **Phase 5** | 18개월~ | + 네이티브 광고/스폰서십 | 고도화 | 건설사/시행사 신규 분양 프로모션 네이티브 광고 |

#### Phase별 상세

**Phase 3 — 프리미엄 구독 서비스 (월 9,900~19,900원)**
- 무료: 일일 폭락/신고가 TOP 5, 기본 금리 정보
- 프리미엄: 전체 랭킹 (TOP 50), 지역별 필터, 폭락/신고가 실시간 알림, 과거 데이터 아카이브, 맞춤 단지 모니터링
- 벤치마크: 부동산지인 프리미엄이 유사 모델로 성공적 운영 중

**Phase 4 — B2B 데이터 API**
- 부동산 유튜버/블로거: 자동화된 차트/데이터를 콘텐츠에 활용 (월 49,000~99,000원)
- 중개업소: 지역별 시세 동향 리포트 자동 생성 (월 29,000원)
- 핀테크 스타트업: 실거래가 분석 데이터 API (월 199,000원~)

**Phase 5 — 네이티브 광고**
- 신규 분양 단지 프로모션을 데일리 리포트 내 "주목할 신규 분양" 섹션으로 자연스럽게 노출
- 건설사/시행사 직접 영업. CPM 50,000~100,000원 (프리미엄 타겟 오디언스)

### 2.9 초기 0→1 전략: 첫 수익까지의 현실적 타임라인

```
[Week 1~2]  MVP 개발 완료
            ├── 메인 대시보드 (폭락/신고가 TOP 5)
            ├── 아파트 단지 상세 페이지 (SEO 핵심)
            └── 구글 애드센스 신청
                    │
[Week 3~4]  콘텐츠 시딩 + 인스타 시작
            ├── 인스타 계정 개설 (@donjupkr), 카드뉴스 매일 1개 발행
            ├── 주요 단지 페이지 300개+ 생성 (SEO 인덱싱 시작)
            └── 애드센스 승인 (보통 1~4주 소요)
                    │
[Month 2]   첫 수익 발생 🎯
            ├── 예상 PV: 월 3,000~5,000 (인스타 + 초기 SEO)
            ├── 예상 애드센스 수익: 월 15,000~40,000원
            ├── 인스타 팔로워: 500~1,000명
            └── 핵심 지표 추적 시작 (GA4 + Search Console)
                    │
[Month 3~4] 금리 계산기 + CPA 연동
            ├── 금리 계산기 페이지 완성
            ├── 핀다/뱅크샐러드 제휴 신청
            ├── 예상 PV: 월 10,000~30,000
            └── 예상 수익: 월 5~15만원
                    │
[Month 5~6] 성장 가속
            ├── SEO 효과 본격화 (3~6개월 후 검색 유입 급증)
            ├── 인스타 팔로워: 5,000~10,000명
            ├── 예상 PV: 월 50,000~100,000
            └── 예상 수익: 월 30~100만원
                    │
[Month 7~12] 안정화 + 프리미엄 전환
             ├── SEO 롱테일 축적 (단지 페이지 1,000개+ × 데일리 리포트 180개+)
             ├── 예상 PV: 월 100,000~500,000
             └── 예상 수익: 월 100~500만원
```

#### 0→1 핵심 원칙

1. **첫 2주는 코딩만 한다.** 완벽한 기능이 아니라 "폭락 TOP 5 + 단지 상세"만 있으면 된다.
2. **SEO가 복리다.** 매일 자동 생성되는 데일리 리포트와 수천 개의 단지 페이지가 3~6개월 후 검색 유입을 폭발시킨다.
3. **인스타는 무료 부스터다.** 카드뉴스는 제작비 0원, 바이럴 확산은 알고리즘이 해준다. "폭락 XX%" 자극적 숫자가 저장/공유를 만든다.
4. **CPA가 진짜 돈이다.** 애드센스는 기본 깔개, CPA 한 건(10,000원)이 애드센스 PV 2,000건 수익과 같다. 금리 계산기 동선 최적화가 수익의 핵심.
5. **서버비 0원이 최강의 해자다.** 월 3만원 벌어도 순이익이다. 경쟁사는 월 수천만원 태우면서 BEP를 고민하지만, 돈줍은 Day 1부터 수익이 곧 이익.

### 2.10 수익 극대화 공식

```
총 수익 = 페이지뷰(PV) × 애드센스 RPM / 1,000
        + 금리계산기 PV × CPA 전환율 × 평균 CPA 단가
        + 부동산 플랫폼 CPC 클릭수 × CPC 단가
        + (Phase 3~) 프리미엄 구독자 × 월 구독료
```

**레버 3가지**:
- **PV 극대화**: 인스타 바이럴(단기) + SEO 롱테일(장기) + 내부 페이지 순환(세션당 2.5+ 페이지)
- **RPM 극대화**: 대출/금리 고단가 키워드 페이지에 광고 집중, 금리 계산기 페이지 RPM 8,000원+ 목표
- **CPA 전환 극대화**: 금리 계산기 결과 직후 "더 낮은 금리 찾기" CTA, 계산 완료 순간이 전환의 골든타임

## 4. 페이지별 기능 정의

### 4.1 메인 대시보드 (`/`)

**목적**: 방문자가 처음 도착하는 곳. 가장 자극적인 데이터로 시선을 잡고, 내부 페이지로 유도한다.

**구성 요소**:

| 영역 | 콘텐츠 | 데이터 소스 |
|------|--------|-----------|
| 히어로 섹션 | "오늘자 서울 폭락 아파트 1위: OO아파트 -35%" | apt_transactions (is_significant_drop) |
| 폭락 TOP 5 | 최고가 대비 하락률 상위 5개 단지 카드 | apt_transactions ORDER BY change_rate ASC |
| 신고가 TOP 5 | 역대 최고가 갱신 단지 카드 | apt_transactions WHERE is_new_high = true |
| 금리 현황 | 기준금리 / COFIX / 전일 대비 변동 뱃지 | finance_rates (최신) |
| 핫스팟 | 거래량 급증 지역 3곳 하이라이트 | apt_transactions GROUP BY region |
| 인기 검색 | 사용자 조회수 기반 인기 단지 TOP 5 | page_views |

**광고 배치**:
- 상단 배너 (728×90): 폭락 TOP 리스트 위
- 네이티브 인피드: 리스트 3번째 항목 아래
- 사이드바 (300×250): 데스크톱 우측

**전환 요소**:
- "내 아파트는 얼마나 떨어졌을까?" → 단지 상세 페이지 유도
- "대출 이자가 궁금하다면?" → 금리 계산기 유도

---

### 4.2 아파트 단지 상세 (`/apt/[region]/[apt-id]`)

**목적**: SEO로 유입되는 핵심 페이지. "OO아파트 실거래가" 검색 시 노출. 체류시간을 늘려 광고 수익 확보.

**구성 요소**:

| 영역 | 콘텐츠 |
|------|--------|
| 단지 요약 카드 | 단지명, 주소, 세대수, 준공연도, 현재 시세 범위 |
| 가격 추이 차트 | 1년/3년/5년 실거래가 라인차트 (최고가 라인 표시) |
| 최고가 대비 현재가 | 게이지 차트: 최고가 100% 기준 현재 위치 시각화 |
| 최근 거래 내역 | 테이블: 거래일, 면적, 층, 가격 (최신순) |
| 평형별 비교 | 59㎡ / 84㎡ / 114㎡ 등 면적별 가격 비교 |
| 주변 단지 비교 | 같은 동 내 다른 단지와 가격 비교 테이블 |

**광고 배치**:
- 콘텐츠 중간 인피드: 차트와 테이블 사이
- 하단 CPA 배너: **"이 아파트, 대출 받으면 월 얼마?"** → 금리 계산기 연결
- 사이드바: 대출/보험 배너

**SEO 설정**:
- `<title>`: "OO아파트 실거래가 - 돈줍"
- `<meta description>`: "OO아파트 84㎡ 최근 거래가 12억 3천만원. 최고가 대비 -15.2%. 전체 거래 내역과 가격 추이를 확인하세요."
- Open Graph 이미지: 가격 추이 차트 자동 생성

---

### 4.3 대출 이자 계산기 (`/rate/calculator`)

**목적**: 서비스 내 **최고 수익 페이지**. 사용자가 직접 금액을 입력하는 행위 = 대출 의향 높음 = 대출 광고 CTR 극대화.

**구성 요소**:

| 영역 | 콘텐츠 |
|------|--------|
| 입력 폼 | 대출 원금 / 금리(현재 기준금리 자동 반영) / 상환 기간 |
| 상환 방식 비교 | 원리금균등 / 원금균등 / 만기일시 상환 3가지 동시 비교 |
| 월별 상환 스케줄 | 접이식 테이블: 매월 원금/이자/잔금 |
| 금리 변동 시나리오 | 슬라이더: 0.25%p 단위로 금리 변동 시 월 이자 변화 시각화 |
| 현재 금리 참고 | 기준금리 / COFIX / 시중은행 금리 밴드 표시 |

**광고 배치** (수익의 핵심):
- 계산 결과 직후: **"더 낮은 금리를 찾고 계신가요?"** CPA 배너 (뱅크샐러드/핀다/토스)
- 사이드바: 대출 상품 비교 배너
- 하단: "내 신용점수 무료 조회" CPA 링크

**전환 최적화**:
- 계산 결과가 나온 직후가 CPA 클릭률이 가장 높은 순간
- "현재 최저 금리 확인하기" 버튼을 결과 바로 아래 배치

**목적**: 매일 1개씩 자동 생성. 연간 365개 페이지 축적으로 SEO 롱테일 효과 극대화.

**구성 요소**:

| 영역 | 콘텐츠 |
|------|--------|

| 거래량 이상 징후 | 거래량이 평소 대비 급증한 지역 |
| 관련 리포트 | 이전/다음 날 리포트 + "이 지역 더 보기" 내부 링크 |

**광고 배치**:
- 콘텐츠 중간 인피드 2개
- 하단 배너
- 관련 리포트 영역 네이티브 광고

## 5. 사용자 플로우 (User Flow)

### 5.1 메인 플로우: 인스타 → 수익 전환

```
[1. 인스타그램 카드뉴스]
   "오늘 강남 XX아파트 2억 폭락!!"
   → 저장/공유 유도 → 바이럴 확산
              │
              ▼
[2. 랜딩: 메인 대시보드 또는 해당 아파트 상세]
   → 폭락/신고가 리스트 확인
   → "내 아파트도 확인해보기" 클릭
              │
              ▼
[3. 아파트 상세 페이지]                    ← 페이지뷰 +1, 광고 노출
   → 가격 추이 차트 확인
   → "이 아파트 대출받으면 월 얼마?" 클릭
              │
              ▼
[4. 금리 계산기 페이지]                    ← 페이지뷰 +1, 고단가 광고 노출
   → 대출 시뮬레이션 실행
   → "더 낮은 금리 찾기" CPA 클릭
              │
              ▼
[5. 제휴 서비스 이동]                      ← CPA 수익 발생
   → 뱅크샐러드/핀다/토스 대출 비교
```

### 5.2 SEO 유입 플로우

```
[1. 네이버/구글 검색]
   "래미안퍼스티지 실거래가"
              │
              ▼
[2. 아파트 단지 상세 페이지]               ← 검색 유입
   → 가격 추이 확인
   → 관련 단지 비교 → 다른 단지 상세       ← 내부 순환 (PV 증가)
   → "대출 이자 계산" 클릭
              │
              ▼
[3. 금리 계산기]                           ← CPA 전환
```

### 5.3 재방문 플로우

```
[1. 습관성 방문]
   매일 아침 메인 대시보드 확인
   → "오늘은 어디가 떨어졌나?"
              │
              ▼
[2. 데일리 리포트 구독]
   → 매일 새로운 리포트 확인
   → 페이지뷰 축적 → 광고 수익 지속
```
