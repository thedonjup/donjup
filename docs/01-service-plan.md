# 돈줍(DonJup) 서비스 기획서

> "부동산과 돈의 흐름을 압축(Zip)해서 보여주는 곳"

---

## 1. 서비스 개요

### 1.1 서비스 정의

| 항목 | 내용 |
|------|------|
| 서비스명 | **돈줍(DonJup)** |
| 도메인 | donjup.com |
| 핵심 가치 | 매일 자동 업데이트되는 극단적 부동산 가격 변동 + 체감형 금리 정보를 한눈에 |
| 한 줄 소개 | "오늘 가장 많이 떨어진 아파트, 오늘 신고가를 찍은 아파트 — 매일 아침 자동 업데이트" |

### 1.2 타겟 사용자 페르소나

| 페르소나 | 특징 | 핵심 니즈 | 유입 경로 |
|----------|------|----------|----------|
| **A. 30대 실수요자** | 내 집 마련 준비 중, 매일 실거래가 확인 | "내가 보는 단지가 지금 비싼 건지 싼 건지" | 인스타 카드뉴스 → 단지 상세 |
| **B. 40대 투자자** | 지역별 거래량/가격 동향으로 투자 타이밍 판단 | "지금 거래가 몰리는 지역이 어디인지" | 검색(SEO) → 지역 동향 |
| **C. 영끌족** | 주담대 이자 변동에 극도로 민감, 금리 0.1%p도 체크 | "금리가 바뀌면 내 이자가 얼마나 바뀌는지" | 인스타/검색 → 금리 계산기 |

### 1.3 핵심 콘텐츠 원칙

사람들은 **평범한 정보에는 반응하지 않는다.** 오직 두 가지 감정만 자극한다:

- **공포(손실)**: "최고가 15억이 9억으로?" → 폭락 아파트 랭킹
- **FOMO(수익)**: "남들 다 떨어지는데 이 단지만 신고가?" → 신고가 갱신 단지

방대한 공공데이터 중 **[단기 폭락/상승 아파트 리스트] + [금리 변동 지표]** 딱 두 가지만 매일 자동으로 발라내어 가공하는 것이 서비스의 핵심 엔진이다.

## 2. 비즈니스 모델 (BM)

### 2.1 수익 구조

```
[인스타그램 카드뉴스] ──바이럴──▶ [웹사이트 유입] ──체류──▶ [광고 노출/클릭] ──▶ 수익
```

### 2.2 수익원 상세

| 순위 | 수익원 | 설명 | 수익 구조 |
|------|--------|------|----------|
| 1 | **구글 애드센스** (메인) | 부동산/대출/금융 카테고리 고단가 디스플레이 광고 | CPM 기반 (페이지뷰당 3~8원) |
| 2 | **CPA 제휴 마케팅** (서브) | 뱅크샐러드/핀다/토스 등 대출 비교 서비스 연결 | 건당 3,000~15,000원 |
| 3 | **부동산 플랫폼 제휴** | 직방/다방/네이버부동산 링크 연결 | CPC 500~2,000원 |
| 4 | **프리미엄 리포트** (향후) | 지역별 심층 분석 리포트 유료화 | 월 9,900원 구독 |

### 2.3 수익 극대화 공식

```
총 수익 = 페이지뷰(PV) × 광고 RPM + CPA 전환 건수 × 건당 수수료
```

- **PV 극대화**: 인스타 바이럴 + SEO 롱테일 + 내부 페이지 순환
- **RPM 극대화**: 부동산/대출 고단가 키워드 콘텐츠에 광고 집중 배치
- **CPA 전환 극대화**: 금리 계산기 결과 직후 "더 낮은 금리 찾기" CTA 배치

## 3. 정보구조 (Information Architecture)

### 3.1 사이트맵

```
돈줍(DonJup)
│
├── / (메인 대시보드)
│   ├── 오늘의 폭락 아파트 TOP 5
│   ├── 오늘의 신고가 아파트 TOP 5
│   ├── 실시간 금리 현황 카드
│   └── 거래량 급증 지역 하이라이트
│
├── /apt (아파트 실거래가)
│   ├── /apt/[region] ─────────── 지역별 아파트 목록
│   └── /apt/[region]/[apt-id] ── 개별 단지 상세 ⭐ SEO 핵심
│
├── /rate (금리/대출 정보)
│   ├── /rate/dashboard ────────── 금리 대시보드 (기준금리 + COFIX 추이)
│   ├── /rate/calculator ───────── 대출 이자 계산기 ⭐ 고단가 광고 핵심
│   └── /rate/history ──────────── 금리 변동 히스토리
│
├── /market (지역별 거래 동향)
│   ├── /market/[region] ───────── 지역별 거래 상세
│   └── /market/heatmap ────────── 거래량 히트맵
│
├── /daily (데일리 리포트)
│   ├── /daily/[date] ──────────── 날짜별 리포트 ⭐ SEO 롱테일
│   └── /daily/archive ─────────── 전체 아카이브
│
└── /about ─────────────────────── 서비스 소개
```

### 3.2 페이지별 역할 요약

| 페이지 | 주요 역할 | 수익 기여 |
|--------|----------|----------|
| 메인 대시보드 | 트래픽 랜딩 + 내부 순환 시작점 | 높은 노출량 (애드센스) |
| 아파트 단지 상세 | SEO 유입 + 체류시간 확보 | CPA 전환 유도 |
| 금리 계산기 | 고의향 사용자 집중 | **최고 수익 페이지** (CPA) |
| 지역별 동향 | 투자자 재방문 유도 | 지역 타겟팅 광고 |
| 데일리 리포트 | SEO 롱테일 축적 (연 365페이지) | 체류시간 기반 노출 |

## 4. 페이지별 기능 정의

### 4.1 메인 대시보드 (`/`)

**목적**: 방문자가 처음 도착하는 곳. 가장 자극적인 데이터로 시선을 잡고, 내부 페이지로 유도한다.

**구성 요소**:

| 영역 | 콘텐츠 | 데이터 소스 |
|------|--------|-----------|
| 히어로 섹션 | "오늘자 서울 폭락 아파트 1위: OO아파트 -35%" | apt_transactions (is_significant_drop) |
| 폭락 TOP 5 | 최고가 대비 하락률 상위 5개 단지 카드 | apt_transactions ORDER BY change_rate ASC |
| 신고가 TOP 5 | 역대 최고가 갱신 단지 카드 | apt_transactions WHERE is_new_high = true |
| 금리 현황 | 기준금리 / COFIX / 전일 대비 변동 뱃지 | finance_rates (최신) |
| 핫스팟 | 거래량 급증 지역 3곳 하이라이트 | apt_transactions GROUP BY region |
| 인기 검색 | 사용자 조회수 기반 인기 단지 TOP 5 | page_views |

**광고 배치**:
- 상단 배너 (728×90): 폭락 TOP 리스트 위
- 네이티브 인피드: 리스트 3번째 항목 아래
- 사이드바 (300×250): 데스크톱 우측

**전환 요소**:
- "내 아파트는 얼마나 떨어졌을까?" → 단지 상세 페이지 유도
- "대출 이자가 궁금하다면?" → 금리 계산기 유도

---

### 4.2 아파트 단지 상세 (`/apt/[region]/[apt-id]`)

**목적**: SEO로 유입되는 핵심 페이지. "OO아파트 실거래가" 검색 시 노출. 체류시간을 늘려 광고 수익 확보.

**구성 요소**:

| 영역 | 콘텐츠 |
|------|--------|
| 단지 요약 카드 | 단지명, 주소, 세대수, 준공연도, 현재 시세 범위 |
| 가격 추이 차트 | 1년/3년/5년 실거래가 라인차트 (최고가 라인 표시) |
| 최고가 대비 현재가 | 게이지 차트: 최고가 100% 기준 현재 위치 시각화 |
| 최근 거래 내역 | 테이블: 거래일, 면적, 층, 가격 (최신순) |
| 평형별 비교 | 59㎡ / 84㎡ / 114㎡ 등 면적별 가격 비교 |
| 주변 단지 비교 | 같은 동 내 다른 단지와 가격 비교 테이블 |

**광고 배치**:
- 콘텐츠 중간 인피드: 차트와 테이블 사이
- 하단 CPA 배너: **"이 아파트, 대출 받으면 월 얼마?"** → 금리 계산기 연결
- 사이드바: 대출/보험 배너

**SEO 설정**:
- `<title>`: "OO아파트 실거래가 - 돈줍"
- `<meta description>`: "OO아파트 84㎡ 최근 거래가 12억 3천만원. 최고가 대비 -15.2%. 전체 거래 내역과 가격 추이를 확인하세요."
- Open Graph 이미지: 가격 추이 차트 자동 생성

---

### 4.3 대출 이자 계산기 (`/rate/calculator`)

**목적**: 서비스 내 **최고 수익 페이지**. 사용자가 직접 금액을 입력하는 행위 = 대출 의향 높음 = 대출 광고 CTR 극대화.

**구성 요소**:

| 영역 | 콘텐츠 |
|------|--------|
| 입력 폼 | 대출 원금 / 금리(현재 기준금리 자동 반영) / 상환 기간 |
| 상환 방식 비교 | 원리금균등 / 원금균등 / 만기일시 상환 3가지 동시 비교 |
| 월별 상환 스케줄 | 접이식 테이블: 매월 원금/이자/잔금 |
| 금리 변동 시나리오 | 슬라이더: 0.25%p 단위로 금리 변동 시 월 이자 변화 시각화 |
| 현재 금리 참고 | 기준금리 / COFIX / 시중은행 금리 밴드 표시 |

**광고 배치** (수익의 핵심):
- 계산 결과 직후: **"더 낮은 금리를 찾고 계신가요?"** CPA 배너 (뱅크샐러드/핀다/토스)
- 사이드바: 대출 상품 비교 배너
- 하단: "내 신용점수 무료 조회" CPA 링크

**전환 최적화**:
- 계산 결과가 나온 직후가 CPA 클릭률이 가장 높은 순간
- "현재 최저 금리 확인하기" 버튼을 결과 바로 아래 배치

---

### 4.4 금리 대시보드 (`/rate/dashboard`)

**목적**: 거시 금리 지표를 한눈에. 영끌족의 습관성 방문을 유도.

**구성 요소**:

| 영역 | 콘텐츠 |
|------|--------|
| 현재 금리 카드 | 기준금리 / 코픽스(신규) / 코픽스(잔액) / CD금리 / 국고채3년 |
| 변동 뱃지 | 각 금리 옆에 ▲▼ 전일/전월 대비 변동 표시 |
| 금리 추이 차트 | 최근 1년 금리 변동 라인차트 (다중 지표 오버레이) |
| 금통위 일정 | 다음 한국은행 금융통화위원회 회의 날짜 카운트다운 |
| 계산기 CTA | "내 대출이자 계산해보기" 버튼 → /rate/calculator |

---

### 4.5 지역별 거래 동향 (`/market/[region]`)

**목적**: 투자자 타겟. "지금 어디서 거래가 활발한지" 흐름을 보여주어 재방문 유도.

**구성 요소**:

| 영역 | 콘텐츠 |
|------|--------|
| 월별 거래건수 차트 | 막대 차트: 최근 12개월 거래량 추이 |
| 평균 매매가 추이 | 라인 차트: 지역 평균 매매가 변동 |
| 거래량 증감 랭킹 | 전월 대비 거래 건수 증감률 TOP 동 |
| 주요 단지 거래 현황 | 해당 지역 내 최근 거래된 단지 리스트 (상세 링크) |

---

### 4.6 데일리 리포트 (`/daily/[date]`)

**목적**: 매일 1개씩 자동 생성. 연간 365개 페이지 축적으로 SEO 롱테일 효과 극대화.

**구성 요소**:

| 영역 | 콘텐츠 |
|------|--------|
| 리포트 제목 | "2026년 3월 22일 부동산 데일리 리포트" |
| 오늘의 요약 | 한 줄 요약: "서울 폭락 거래 3건, 신고가 1건, COFIX 0.02%p 하락" |
| 폭락 TOP 3 | 당일 하락률 상위 3개 거래 상세 |
| 신고가 단지 | 당일 신고가 갱신 거래 전체 |
| 금리 변동 | 전일 대비 금리 변동 사항 |
| 거래량 이상 징후 | 거래량이 평소 대비 급증한 지역 |
| 관련 리포트 | 이전/다음 날 리포트 + "이 지역 더 보기" 내부 링크 |

**광고 배치**:
- 콘텐츠 중간 인피드 2개
- 하단 배너
- 관련 리포트 영역 네이티브 광고

## 5. 사용자 플로우 (User Flow)

### 5.1 메인 플로우: 인스타 → 수익 전환

```
[1. 인스타그램 카드뉴스]
   "오늘 강남 XX아파트 2억 폭락!!"
   → 저장/공유 유도 → 바이럴 확산
              │
              ▼
[2. 랜딩: 메인 대시보드 또는 해당 아파트 상세]
   → 폭락/신고가 리스트 확인
   → "내 아파트도 확인해보기" 클릭
              │
              ▼
[3. 아파트 상세 페이지]                    ← 페이지뷰 +1, 광고 노출
   → 가격 추이 차트 확인
   → "이 아파트 대출받으면 월 얼마?" 클릭
              │
              ▼
[4. 금리 계산기 페이지]                    ← 페이지뷰 +1, 고단가 광고 노출
   → 대출 시뮬레이션 실행
   → "더 낮은 금리 찾기" CPA 클릭
              │
              ▼
[5. 제휴 서비스 이동]                      ← CPA 수익 발생
   → 뱅크샐러드/핀다/토스 대출 비교
```

### 5.2 SEO 유입 플로우

```
[1. 네이버/구글 검색]
   "래미안퍼스티지 실거래가"
              │
              ▼
[2. 아파트 단지 상세 페이지]               ← 검색 유입
   → 가격 추이 확인
   → 관련 단지 비교 → 다른 단지 상세       ← 내부 순환 (PV 증가)
   → "대출 이자 계산" 클릭
              │
              ▼
[3. 금리 계산기]                           ← CPA 전환
```

### 5.3 재방문 플로우

```
[1. 습관성 방문]
   매일 아침 메인 대시보드 확인
   → "오늘은 어디가 떨어졌나?"
              │
              ▼
[2. 데일리 리포트 구독]
   → 매일 새로운 리포트 확인
   → 페이지뷰 축적 → 광고 수익 지속
```
