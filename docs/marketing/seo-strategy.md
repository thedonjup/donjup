## 1. 현재 SEO 상태 진단

### 1.1 잘 되어 있는 부분

| 항목 | 상태 | 비고 |
|------|------|------|
| `<html lang="ko">` | O | 한국어 언어 설정 완료 |
| `metadataBase` | O | `https://donjup.com` 설정됨 |
| `title` template | O | `%s \| 돈줍` 패턴 적용 |
| `description` | O | 메인 페이지 및 주요 페이지 모두 설정 |
| `keywords` | O | 메인/지역별/전월세/트렌드 등 설정 |
| `robots.txt` | O | `/api/` disallow, sitemap 연결 |
| `sitemap.xml` | O | 정적 페이지 + 시도/시군구 동적 생성 |
| OG 태그 | O | locale, siteName, title, description |
| Twitter Card | O | summary_large_image |
| OG 이미지 | O | `opengraph-image.tsx` 동적 생성 (1200x630) |
| JSON-LD (홈) | O | WebSite + SearchAction 스키마 |
| JSON-LD (아파트) | O | RealEstateListing 스키마 |
| PWA manifest | O | `/manifest.json` |
| GA4 | O | Google Analytics 연동 |
| AdSense | O | 연동 완료 |
| 접근성 | O | skip-link, aria-label |
| breadcrumb | O | 아파트 상세 페이지에 구현 |
| generateStaticParams | O | 시군구 페이지 정적 생성 |

### 1.2 개선이 필요한 부분

| 항목 | 현재 | 문제점 |
|------|------|--------|
| sitemap에 아파트 상세 누락 | `/apt/{region}/{slug}` 미포함 | 가장 중요한 롱테일 페이지가 색인 안 됨 |
| sitemap에 /today, /new-highs 누락 | 미포함 | 매일 갱신되는 핵심 페이지 |
| sitemap에 /trend, /themes, /compare 누락 | 미포함 | 주요 콘텐츠 페이지 |
| JSON-LD 대부분 페이지 없음 | 홈/아파트만 | 금리, 트렌드, 시장 등 구조화 데이터 없음 |
| BreadcrumbList 구조화 데이터 없음 | HTML만 있음 | JSON-LD BreadcrumbList 없음 |
| 네이버 서치어드바이저 미확인 | 미등록 추정 | 한국 검색 50%+ 점유율 미활용 |
| OG 이미지 정적 | 모든 페이지 동일 | 개별 페이지별 동적 OG 이미지 필요 |
| canonical URL 미설정 | 없음 | 중복 콘텐츠 이슈 가능 |
| hreflang 없음 | 단일 언어 | 향후 다국어 고려시 필요 |
| /compare, /themes 메타데이터 없음 | 미확인 | metadata export 누락 가능 |
| 데일리 리포트 개별 페이지 sitemap 미포함 | `/daily/{date}` 미포함 | 날짜별 아카이브 색인 안 됨 |
| robots.txt에 /dam/ 미차단 | admin 페이지 노출 | 관리자 페이지가 색인될 수 있음 |

## 3. 경쟁사 SEO 분석

### 3.1 경쟁사 현황

| 경쟁사 | 월간 방문자(추정) | 핵심 전략 | 강점 |
|--------|------------------|-----------|------|
| **호갱노노** | 300만+ | 단지별 SEO, 리뷰/커뮤니티 | 브랜드 파워, 단지 리뷰, 지도 기반 |
| **아실** | 200만+ | 실거래가 데이터, 앱 중심 | 모바일 앱 DAU, 알림 기능 |
| **직방** | 500만+ | 매물 + 시세, 광고 | 매물 연동, 방대한 데이터 |
| **네이버부동산** | 1,000만+ | 네이버 생태계 통합 | 네이버 검색 1위 노출, 매물 통합 |
| **부동산뱅크** | 50만+ | 시세 데이터 | 오랜 업력, KB시세 연동 |

### 3.2 경쟁사 상위 노출 키워드 패턴

```
호갱노노: "{아파트명} 시세", "{아파트명} 리뷰", "{지역} 아파트 추천"
아실:     "{아파트명} 실거래가", "{지역} 아파트 매매", "아파트 분석"
직방:     "{아파트명} 매물", "{지역} 아파트", "원룸 월세"
네이버:   "{아파트명}", "{지역} 부동산", "아파트 시세"
```

### 3.3 돈줍의 차별화 기회

돈줍은 경쟁사가 약한 다음 영역에서 차별화 가능:

1. **"폭락" 키워드 독점** - 경쟁사 중 폭락 순위를 전면에 내세우는 곳 없음
2. **"신고가 랭킹"** - 실시간 신고가 경신 알림은 독자적 포지션
3. **"가격 하락률 순위"** - 데이터 기반 하락 분석은 유일
4. **금리 + 부동산 통합** - 금리와 시세를 하나에서 보는 플랫폼 드묾
5. **데일리 리포트** - 매일 자동 생성되는 시장 분석 리포트

## 4. 페이지별 SEO 개선 액션 리스트

### 4.1 [긴급] sitemap.xml 확장

**파일:** `src/app/sitemap.ts`

추가해야 할 페이지:
- `/today` - 오늘의 거래 (priority: 0.9, daily)
- `/new-highs` - 오늘의 신고가 (priority: 0.9, daily)
- `/trend` - 트렌드 (priority: 0.8, daily)
- `/themes` - 테마 컬렉션 (priority: 0.7, weekly)
- `/compare` - 단지 비교 (priority: 0.7, monthly)
- `/about` - 서비스 소개 (priority: 0.3, monthly)
- `/apt/{region}/{slug}` - 전체 아파트 단지 상세 (priority: 0.9, daily) **[가장 중요]**
- `/daily/{date}` - 데일리 리포트 아카이브 (priority: 0.6, daily)

> 아파트 단지 URL이 sitemap에 없으면 구글이 크롤링할 기회가 극히 적음.
> Supabase에서 apt_complexes 전체 slug를 조회하여 동적으로 sitemap에 포함해야 함.
> 50,000개 이상이면 sitemap index로 분할 필요.

### 4.2 [긴급] robots.txt 개선

**파일:** `src/app/robots.ts`

```typescript
// 추가 필요
disallow: ["/api/", "/dam/", "/profile/"]
```

관리자 패널(`/dam/`)과 개인 프로필(`/profile/`)은 검색 노출 차단 필요.

### 4.3 [중요] 홈페이지 (/)

현재 상태: 양호
개선사항:
- [ ] `<h1>` 태그에 핵심 키워드 "아파트 실거래가 폭락 순위" 포함 확인
- [ ] FAQ 섹션 추가 (FAQPage 구조화 데이터 + "아파트 폭락이란?", "실거래가 조회 방법" 등)

## 5. 구조화 데이터(JSON-LD) 추가 계획

### 5.1 현재 구현된 스키마

| 페이지 | 스키마 타입 | 상태 |
|--------|------------|------|
| / (홈) | WebSite + SearchAction | O |
| /apt/{region}/{slug} | RealEstateListing | O |

### 5.2 추가 필요한 스키마

#### (1) BreadcrumbList - 전체 페이지 공통

```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    { "@type": "ListItem", "position": 1, "name": "홈", "item": "https://donjup.com" },
    { "@type": "ListItem", "position": 2, "name": "서울", "item": "https://donjup.com/market/seoul" },
    { "@type": "ListItem", "position": 3, "name": "강남구", "item": "https://donjup.com/market/seoul/11680" }
  ]
}
```

적용 페이지: /market/{sido}, /market/{sido}/{sigungu}, /apt/{region}/{slug}

#### (2) Organization - 홈페이지

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "돈줍",
  "alternateName": "DonJup",
  "url": "https://donjup.com",
  "logo": "https://donjup.com/logo.svg",
  "sameAs": ["https://instagram.com/donjup.official"]
}
```

#### (3) FAQPage - 홈, 아파트 상세

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "아파트 실거래가 폭락이란?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "아파트 실거래가 폭락은 해당 단지의 역대 최고 거래가 대비 현재 거래가가 크게 하락한 것을 의미합니다. 돈줍에서는 최고가 대비 20% 이상 하락을 폭락, 30% 이상을 대폭락으로 분류합니다."
      }
    }
  ]
}
```

#### (4) ItemList - 폭락 순위, 신고가, 오늘 거래

```json
{
  "@context": "https://schema.org",
  "@type": "ItemList",
  "name": "전국 아파트 폭락 순위",
  "numberOfItems": 10,
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "url": "https://donjup.com/apt/seoul/11680-래미안-퍼스티지",
      "name": "래미안 퍼스티지 -15.2%"
    }
  ]
}
```

#### (5) FinancialProduct - 금리 페이지

```json
{
  "@context": "https://schema.org",
  "@type": "FinancialProduct",
  "name": "주택담보대출 금리 비교",
  "provider": { "@type": "Organization", "name": "돈줍" },
  "description": "주요 은행별 주택담보대출 금리 비교"
}
```

#### (6) Dataset - 트렌드 페이지

```json
{
  "@context": "https://schema.org",
  "@type": "Dataset",
  "name": "전국 아파트 거래 통계",
  "description": "국토교통부 실거래가 공개시스템 기반 전국 아파트 거래 통계",
  "provider": { "@type": "Organization", "name": "돈줍" },
  "temporalCoverage": "2006/..",
  "spatialCoverage": "대한민국"
}
```

#### (7) SoftwareApplication - PWA용 (선택)

```json
{
  "@context": "https://schema.org",
  "@type": "WebApplication",
  "name": "돈줍",
  "operatingSystem": "All",
  "applicationCategory": "FinanceApplication",
  "offers": { "@type": "Offer", "price": "0", "priceCurrency": "KRW" }
}
```

### 5.3 구현 우선순위

1. BreadcrumbList (아파트 상세 + 지역별) - 1주차
2. Organization (홈) - 1주차
3. ItemList (홈 폭락/신고가 리스트) - 2주차
4. FAQPage (홈 + 아파트 상세) - 2주차
5. FinancialProduct (금리) - 3주차
6. Dataset (트렌드) - 3주차

## 6. 네이버 최적화 체크리스트

한국 검색 시장에서 네이버는 여전히 40~50% 점유율을 가지므로, 네이버 SEO는 필수.

### 6.1 네이버 서치어드바이저 (최우선)

- [ ] https://searchadvisor.naver.com 접속 및 사이트 등록
- [ ] HTML 메타태그 또는 파일 업로드로 소유 확인
- [ ] 사이트맵 제출 (sitemap.xml)
- [ ] RSS 피드 제출 (데일리 리포트용)
- [ ] 크롤링 요청
- [ ] 로봇 확인 (robots.txt 정상 인식 여부)
- [ ] 색인 현황 모니터링

### 6.2 네이버 웹마스터 가이드라인 준수

- [ ] `<meta name="naver-site-verification" content="...">` 태그 추가
- [ ] 네이버봇 크롤링 허용 확인 (robots.txt에 Naver 차단 없는지)
- [ ] 모바일 최적화 확인 (네이버 모바일 검색 우대)
- [ ] 페이지 로딩 속도 3초 이내 확인
- [ ] 중복 콘텐츠 없는지 canonical 설정

### 6.3 네이버 콘텐츠 연동 전략

#### 네이버 블로그 운영
- [ ] 공식 네이버 블로그 개설 (블로그명: "돈줍 - 부동산 실거래가 분석")
- [ ] 주 3회 이상 포스팅 (데일리 리포트 요약, 지역 분석, 금리 해설)
- [ ] 포스팅 내 donjup.com 링크 삽입 (백링크 효과)
- [ ] 포스팅 키워드 전략:
  - 매일: "오늘의 아파트 폭락 TOP 10" / "오늘의 신고가 아파트"
  - 주간: "{지역} 아파트 시세 주간 분석" / "금리 변동 주간 리뷰"
  - 월간: "전국 아파트 거래량 월간 리포트"
- [ ] 이미지 ALT 태그에 키워드 포함
- [ ] 블로그-웹사이트 상호 링크

#### 네이버 카페 활동
- [ ] 부동산 관련 인기 카페 참여 (부동산스터디, 아기곰의 부동산, 월급쟁이 부동산 등)
- [ ] 유용한 분석 정보 공유 + donjup.com 자연스러운 언급
- [ ] 자체 카페 운영 검토 (커뮤니티 기반 트래픽)

#### 네이버 지식iN
- [ ] 부동산 관련 질문에 전문 답변 + 출처 링크
- [ ] "아파트 시세 확인 방법", "실거래가 조회" 등 자주 묻는 질문 답변

### 6.4 네이버 View(블로그/카페) 상위 노출 전략

네이버 검색 결과에서 VIEW 탭은 블로그/카페 콘텐츠를 보여주며, 많은 사용자가 이곳에서 정보를 얻음.

- 블로그 포스팅 제목에 검색 키워드 정확히 포함
- 본문 2,000자 이상, 이미지 5장 이상
- 댓글/공감 유도 (engagement 신호)
- 블로그 지수 관리 (정기 포스팅, 이웃 소통)

### 6.5 네이버 스마트플레이스 (해당시)

- 사업자 등록이 있다면 네이버 스마트플레이스 등록 검토
- 서비스 카테고리: 부동산 정보 서비스

## 부록: 빠른 실행 코드 참고

### A. sitemap.ts 확장 (아파트 상세 추가)

```typescript
// src/app/sitemap.ts 에 추가
// apt_complexes 테이블에서 전체 slug 조회
const { data: complexes } = await supabase
  .from("apt_complexes")
  .select("slug,region_code,updated_at")
  .limit(50000);

const aptPages: MetadataRoute.Sitemap = (complexes ?? []).map((c) => {
  const regionCode = c.region_code?.substring(0, 2) ?? "11";
  // regionCode → sido slug 매핑 필요
  return {
    url: `${baseUrl}/apt/${regionCode}/${c.slug}`,
    lastModified: c.updated_at ? new Date(c.updated_at) : new Date(),
    changeFrequency: "daily" as const,
    priority: 0.8,
  };
});
```
