```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    { "@type": "ListItem", "position": 1, "name": "홈", "item": "https://donjup.com" },
    { "@type": "ListItem", "position": 2, "name": "시세", "item": "https://donjup.com/market" },
    { "@type": "ListItem", "position": 3, "name": "서울", "item": "https://donjup.com/market/seoul" },
    { "@type": "ListItem", "position": 4, "name": "강남구", "item": "https://donjup.com/market/seoul/11680" }
  ]
}
```

### 1.5 기술적 SEO 체크리스트

#### 필수 (P0)

- [ ] **sitemap.xml 확장**: `/today`, `/new-highs`, `/trend`, `/themes`, `/compare`, `/daily/{date}` 추가
- [ ] **sitemap에 아파트 상세 페이지 포함**: `/apt/{region}/{slug}` (수천 페이지)
- [ ] **canonical URL 설정**: 모든 페이지에 `<link rel="canonical">` 추가
- [ ] **robots.txt 수정**: `/dam/` (관리자 페이지) disallow 추가
- [ ] **페이지 로딩 속도**: Core Web Vitals (LCP < 2.5s, FID < 100ms, CLS < 0.1) 달성
- [ ] **모바일 우선 인덱싱**: 모바일에서 모든 콘텐츠 접근 가능 확인
- [ ] **내부 링크 구조 정비**: 폭락 카드 → 단지 상세 클릭 연결 (현재 미구현)
- [ ] **404 페이지 SEO**: 검색 유도 + 인기 페이지 링크 포함 (현재 구현됨)
- [ ] **페이지별 고유 H1 태그**: 중복 H1 없는지 점검

#### 중요 (P1)

- [ ] **이미지 최적화**: next/image 활용, WebP 포맷, alt 텍스트 전체 점검
- [ ] **페이지별 동적 OG 이미지**: 현재 전 페이지 동일 → 개별 페이지별 동적 생성
- [ ] **인터널 링크 자동화**: 관련 단지 추천, 같은 지역 다른 단지 링크
- [ ] **페이지네이션 SEO**: `rel="next"`, `rel="prev"` 또는 무한 스크롤 SEO 대응
- [ ] **구조화 데이터 테스트**: Google Rich Results Test 통과 확인
- [ ] **크롤링 예산 최적화**: 불필요한 페이지 noindex (검색 결과 페이지 등)
- [ ] **HTTPS 리다이렉트**: HTTP → HTTPS 301 리다이렉트 확인
- [ ] **URL 정규화**: trailing slash 통일

#### 개선 (P2)

- [ ] **hreflang 태그**: 향후 다국어 대응 시 준비

**핵심 원칙**:
- 첫 방문에서 가치를 즉시 보여줌 (로그인 없이 모든 기능 이용 가능)

5. **A/B 테스트**: 광고 위치별 RPM 측정 → 최적 배치 도출
